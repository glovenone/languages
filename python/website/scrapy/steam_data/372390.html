About This Software

D3DGear - A fast game recording and live streaming software for PC.

D3DGear is a fast game recording software for PC. Users will be able to record their gameplay without slowing down the game. Our software has minimal performance impact, and won’t cause games to lag or drop large frame rate like most recording software.

D3DGear will quickly stream to twitch.tv without effecting game performance, giving users the chance to show off their skills. D3DGear streaming function is fast and easy to use. D3DGear works with even slow upload speeds. It is fully compatible with twitch.v, hitbox.tv, and other game streaming websites.

Key recording features:

  • Display game FPS HUD.
  • Record DirectX 8/9/10/11/12, OpenGL and Mantle game natively
  • Record Windows 7 Aero desktop and Windows 8/8.1 desktop
  • Allow custom movie resolution, frame rate, and encoding quality
  • Supports Fast MPEG-4v2, Fast MJPEG, Windows Media Video 9 and Huffyuv Lossless software codec
  • Supports AMD APP, NVIDIA NVENC, and Intel QuickSync GPU hardware codec
  • Supports multi-thread encoding, allows customizable number of encoding threads
  • Save video as AVI, AVI2, and WMV format
  • Record and mix game audio, allows adjust volume level of game audio
  • Record and mix audio commentary, allows adjust volume level of audio commentary
  • Support MP3 audio encoder
  • Allow save audio commentary in separate second audio track
  • Supports webcam and media file overlays

Visit the D3DGear website at http://www.d3dgear.com for detailed information about the software features, and to download the free trial.

    • OS: Windows XP SP3

    Recommended:

I first made use of D3D Gear back in 2012, when I was working on a Let's Play series of "The Witcher: Enhanced Edition." I would dabble in it, watch it develop for a while, and give it another chance periodically as the years went by, but recently I decided to give it another go with modern games and see how the new codecs and new features stacked up. So far, I'm pleased with the results.

D3D Gear may lack a lot of the customization you could get with programs like XSplit and OBS. In many ways, I'd compare it more closely with Bandicam (which is a paid recording tool). Yet, what you do get control over allows for some great flexibility in the quality of content you can produce.

One thing that I thought was really cool is that with hardware encoding (H.264), the latest (as of this review) build of D3D Gear actually does something the other programs that do hardware encoding can't: It allows for video encoding in ultra-HD resolutions, beyond 1080p60. In fact, I've produced some very high quality videos at resolutions like 2560x1080 and had no problems whatsoever with H.264 at 60 FPS.

The program is, though, a bit less user-friendly than you'd hope. You will have to manually go in and tinker with things like webcam placement and resolution, for example. Also, right now there is no direct bit-rate control...You get three quality presets, and I've noticed with very intensive games, there is some video file output lag (no matter the quality of the drive recorded to) that result because the bitrate is huge (on very high resolutions, it's FRAPS-quality bitrate, in the area of 200,000 kbps or higher). To me, this is excessive. For normal 1920x1080 at 60 FPS, setting a limit like 20,000 kbps would be plenty for YouTube. So, the ability to set a "max bitrate" would certainly be nice.

Overall, though, the program has great features. Keep in mind though that as of this build, the back end that hooks into games causes conflicts with programs like OBS and XSplit. There's simply no solution other than totally uninstalling D3D Gear, so keep that in mind if you wonder why certain functionalities for tools like that fail.

Still, as of the build I currently have, with a 5 out of 10 being "Average," I'd rate this software at:

VERY GOOD...... 8 / 10
