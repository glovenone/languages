A fun game to learn typing, Typer Shark! Deluxe is a fun game. Diving to shipwrecks in order to find valuable treasures (and fending off hungry sealife by electro shocks generated from vicious typing) becomes repetitive after a while, but it is definitely one of the more fun ways of learning how to touchtype.

The problem I have with Typer Shark is the fact that it can't be properly shut down. It will run on in the background and log more and more time, unless you reboot the computer. That is a bit annoying, and it would be really nice if that would be gotten rid of in a future update (though I suppose this won't really happen).

Since I don't feel like I can learn much more from this game, I will clock it as "played through" and turn towards other games.
