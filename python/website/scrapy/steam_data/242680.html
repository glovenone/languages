Update #98 - Thronebutt - February 21st, 2016

It's been a while, but update 98 finally seems ready. While we've done lots of testing on this build over the last few weeks, do note that the changes to this build are 'deep' enough that we can't guarantee it's 100% stable. Over the next few weeks we'll work on update 99, which will fix any issues that come up as 98 rolls out.

Most of Update 98 is balancing and fixes to issues introduced in 97, but the biggest change is the re-introduction of the Thronebutt meta-leaderboards. Nuclear Throne now directly interfaces with the site, and that means you can keep track of your Daily and Weekly performance there.

Thanks for the patience, and let us know what you think!

    Features
  • Thronebutt.com integration!

    Balancing
  • Heavy Slugger Buff
  • Jungle flies now have a chance of dropping maggots
  • Lightning Crystals are now more aggressive
  • Ultra Bolt Damage Buff
  • Some minor Steroids changes (now starts with bullets, starting ammo increase)
  • Huge things can no longer be revived (bosses, etc.)

    Fixes
  • Fixed some retroactive achievements
  • Removed Daily/Weekly from DRM-Free builds
  • Coop fixes
  • IDPD Van Depth fixes
  • Several No Portal Bugs fixed
  • Less freeze frames on Ally death
  • Skeleton now loses mutations properly
  • Lightning can now escape certain levels
  • Mom attacks with toxic again
  • YV purgatory bug fixed
  • Lots of crashes fixed
  • Van infinite loop fixed

    Misc
  • Collision optimalisations
  • Lightning optimalisations
  • Melting scorchmarks depth issues resolved
  • Loadout guns no longer sound like chicken sword and vice versa

As a fun side-note, community member lietu has been working on Throne2Speech, which sort of narrates your Nuclear Throne run using the new Thronebutt. It's still in beta, so it might not be perfect still. We're looking forward to what people create with the new stream key and Thronebutt integrations!

About This Game

Nuclear Throne is a post-apocalyptic roguelike-like top-down shooter. Not 'the final hope of humanity' post-apocalyptic, but 'humanity is extinct and mutants and monsters now roam the world' post-apocalyptic. Fight your way through the wastelands with powerful weaponry, collecting radiation to mutate some new limbs and abilities. All these things and more you could do if only you were good at this game. Can you reach the Nuclear Throne?

Would I recommend this game? Absolutely, if you would like a shooter game that has a lot of character and a lot of replayability. It has a lot of playable characters that can drastically alter how you play, and tons of weapons you can try, a lot of which are fun.

However that recommendation comes with some caveats, as this is not an entirely fair or skill-based game. It's better at it than say, Binding of Isaac, but you are still going to suffer a lot of cheap deaths. Things exploding on you, guys charging in from off-screen, or just getting bad weapons or bad spawns (I'm aware Hammerhead is a thing, but I feel if you need a specific perk to show up to counteract a situation, that's still a flaw even if there's a fix for it - plus there are still enemies that don't really much about what cover you have).

The game's not entirely balanced, either. There's weapons you wouldn't want to use, like the Toxic weapons, which are impractical and can hurt you since there's no way to negate toxic damage, or the Screwdriver, by far the worst melee weapon in the game which is only useful for one secret; seeing it drop after you can get that secret feels like a joke on you.

It also feels like there's a big disparity in mutations, as well, since there are ones like Boiling Veins, which protects you from dying to explosions (and this game has a LOT of explosions), and Strong Spirit, which gives you a mulligan from death once per stage; really practical things as you go through the game. Then there's say, Lucky Shot, which can give you ammo of a random type when you kill an enemy - except that you usually won't find yourself switching weapons all that often, and the game tends to naturally drop ammo for stuff you don't have anyway. Or Extra Feet, which makes you faster (something one of the character does already), and negates the effect of terrain...but there are only instances of that in two areas, one of which is only a single stage long. I understand this is meant to encourage different builds, but there's a good reason some of these are considered 'core' mutations and others aren't.

I should say this isn't the case for the characters, though, as every one of them feels distinct and appeals to a different kind of player. You want to be good on Ammo, pick Fish. You want to be careful, Crystal or Eyes. Pure damage? Steroids or YV. Want the game to abuse you like a dog? Go Melting.

I don't know how many of the reviews bring this up, but I also love the game's sound. Pretty much every enemy has a bunch of unique soundbytes for attacks they use, which is helpful for reacting, as well as when they die, which is useful for knowing when you've killed something offscreen (and is also immensely satisfying for several foes - looking at you, Lil' Hunter). This isn't always foolproof though, as sometimes a bunch of sounds can happen at the same time.

I hope I haven't implied I dislike the game by criticizing it, because if I genuinely didn't like it I wouldn't have played as long as I have. It's a great game. But playing it is pretty much tough love, and you'll need to keep that in mind a lot.
