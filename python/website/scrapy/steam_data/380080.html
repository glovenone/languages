27 June

Update - Japanese Localization Now Available!

Cakewalk is happy to announce our Japanese localization is now available on Steam!

This localization covers:

  • SONAR UI
  • SONAR Start Page
  • SONAR Local Help
  • SONAR Readme
  • Help Module
  • Classic LP EQ/Compressor Local Help
  • Cakewalk Publisher Local Help
Users who have already installed SONAR and would like to switch languages can do the following:
  • Right-Click on SONAR in your Steam Library
  • Go to Properties
  • Go to the 'Language' Tab
  • Select your preferred language
  • Switch to the 'Local Data' Tab
  • Click the [Verify Integrity of Application Cache...] button.
  • Allow Steam to finish the process, then launch SONAR.

This update applies to SONAR Steam Edition, Professional, and Platinum

20 June

SONAR 2016.06 - Now Available!



Update 6 is now available!

The new features for this release are as follows:

Russian Localization

With update 6 comes our first pass at Russian Localization! Localization is limited to the following parts of the application right now.
  • Main User Interface
  • Plugin Manager
  • VST Scanner
  • Registration Applet
  • Start Page
  • Readme
SONAR is a large application and the localization process can be quite daunting. A few things to take note of:
  • The Help Module will still display in English at this time. This is similar to a few of our other localization options (Italian etc)
  • As the user base expands in these regions, we'll expand the scope and completeness of localization.
  • If you see any issues in the translation/display of the Russian localization, please let us know in the Technical Support Forum!

Smart Swipe
  • With a single click and swipe, you can now engage or disengage multiple track controls across multiple tracks in one gesture. Solo or mute an entire drum set, all your guitars, or all your background vocals. With Smart Swipe you can quickly enable/disable controls on multiple tracks by simply clicking a button in one track, then dragging across adjacent tracks without releasing the mouse button. Only buttons that match the clicked button’s state will change. This is possible in both the Track and Console view using a variety of controls including lanes.

TH3 3.2.2
  • 4x faster plugin loading time
  • Sound chain overview panel for easier navigation
  • Midi Learn functionality now works properly
  • Cabinet IR in now included
  • VST3 Category is now assigned properly

Theme Fixes:
  • Tungsten: Track view & Console view track name labels text were different colors
  • Tungsten: Track view drag move indicator didn't match theme
  • Tungsten: TV meter label & tick mark colors improved
  • Tungsten: Video pane UI updated
  • Tungsten: Navigator pane scroll buttons updated
  • Tungsten: Step Sequencer UI updated
  • Tungsten: PRV Track Pane Text/Focus match the rest of theme
  • Tungsten: Fixed drawing artifacts at bottom of ProChannel
  • Tungsten: "X" to close view tabs in multi-doc are nearly invisible
  • Tungsten: Pro Channel FX Chain UI updated
  • Tungsten: MSR button handling improved when archived
  • Mercury: Tabs in Track view and Console view UI improved
  • Fixed unreadable menu items on systems customized in X3.

Fixes & Enhancements:
  • Resolved crash adding Instrument Tracks in projects that contain large amounts of edits
  • Resolved issue where Global Solo/Unsolo breaks signal flow with Patch Point Routings
  • Fixed Nowtime placement not follow snap settings (or Aim Assist) in automation lanes
  • Bounce to track settings now persist correctly
  • Fixed drawing artifacts when moving Console View separator
  • Resolved issue where WAI was stuck in Console view
  • Fixed broken documentation link with "MIDI Device Failure" dialog
  • Resolved a possible crash when deleting Take Lane

About This Software

Choose the creative experience only SONAR offers: advanced technology, effortless workflow, and an inviting interface that amplifies inspiration. Fueled by over 25 years in the relentless pursuit of innovation, SONAR is re-inventing the modern recording studio.

SONAR Steam Edition is tailored for aspiring content creators and game sound designers. The program includes features like unlimited audio and MIDI tracks, VST3 support, virtual instruments, effects, editing and mixing tools, step sequencer, Matrix View, and much more.

2016 STEAM Season Pass

Enjoy Cakewalk's rolling updates to 12/31/16. After that time you can keep and use SONAR forever. Continued updates will be available via future passes.

  • Dota 2 Music Pack Template

  • 50 Sound FX tailored for video game design (expandable with DLC packs)

  • 8-bit video game sounds that can be triggered via the Matrix View
  • Includes 9 synths and 20 audio effects

  • Windows 7/8/10 support
  • Windows 8/10 Multi-touch support
  • Project Tutorials
  • Mixing and Mastering Templates

    Minimum:
    • OS: Windows 7, 8, or 10 (32- or 64-bit)
    • Processor: 2.6GHz Intel or AMD multi-core processor
    • Memory: 4 GB RAM
    • Graphics: 1280x800 (1920 x 1080+ recommended)
    • Storage: 12 GB available space
    • Sound Card: WDM/KS, WASAPI, or MME compatible device
    • Additional Notes: Connection required for downloads, activation, and publishing features.
    Recommended:
    • Storage: 30 GB available space
    • Sound Card: ASIO compatible

Posted: 30 April
I've been a diehard Sonar user all the way from Cakewalk Pro Audio 9. This is my 3rd Sonar product via Steam, and it's by far the best iteration. And because the payment system was changed right in the middle of 2015, I was given a redemption code. (Super awesome!)

I had 1,363 hours logged into Sonar (2015), so I'm comparing my first hour or so in the new program and these are my thoughts:

This release is noticeably faster and more stable. The PRV is much more responsive and doesn't lag at all like it used to when there were a lot of notes from many different instruments displaying. This also carries over to the main track view where things just move and feel better.

Optimizations have also been made as the program loads up VSTs quicker and doesn't seem to take as long with exporting WAV or MP3 files. Overall seems to be smoother.

The new version also comes with a new instrument called Ultra Analog Session 2. I'd imagine that if you were into the 'new retro wave', that you'd be able to use this as well as the packaged Z3TA+ 1.5 and make some amazing music.

I think Sonar is really starting to come out into its own again. I'm very happy with how the PRV is coming back around because it wasn't too long ago when Cakewalk had a very solid grip on the PRV/MIDI programming side of things.

One last thing I'd like to mention is that I'm very impressed with how much of a presence Seth Kellogg has had on the Steam forums helping people. I hope this hasn't gone unnoticed by his superiors because to have a direct connection to the product when you need something is vital. Seth has helped me a bunch answering my stupid questions as well as taking my feedback.

I look forward to another 1,000+ hours in Sonar Platinum!

*EDIT*
I'd like to mention that I use Windows 7 Home Premium 64 bit with an AMD FX-8350 with 8GB of RAM. I'm regularly using multiple instances of Konakt 5 with many Waves plugins. Stable and solid as a rock!
*/EDIT*
