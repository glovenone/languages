About This Game

Odyssey Reborn is a free 2d top-view online RPG that is set within a medieval setting with SNES like graphics. There are several player classes available, Knight, Mage, Crusader, Necromancer, Rogue, and Cleric. The main playerbase focuses on PVP but the game is also very well setup for PvE action.
Odyssey is a fairly basic hack and slash ORPG (Online Role Playing Game). It is designed to run on lower end computers, some as slow as 66mhz. Instead of focusing highly on visual, and aural elements, Odyssey tends to focus on gameplay value, and replay-ability.

This is an honest review from someone who has played this game since it's initial release nearly 18 years ago. It is an original Visual Basic 6 game that was originally created as a programming class assignment. The goal was to show that an online RPG was able to be created on the simple programming language. Little did Justin (Bugaboo) know, he would create an insanely addictive game. Odyssey Reborn is an attempt to maintain the longest running VB6 RPG on the entire internet. It is the Grandfather of 2d top view VB6 RPGs.

Like most small games, it is only as fun as the people who play it. At times, the community can be a bit toxic; however, if you have a few friends who start playing this game with you at the game time... it can be an absolute blast. The dungeons are fun and PVP can be addictive if it is occuring.

The biggest difficulty is getting the game to run properly as Microsoft no longer installed the dependencies that support games programmed in Visual Basic 6. This is why there are many error reports when installs occur.

Ultimately, I recommend trying out this game. Find a way to obtain as much enjoyment from the game as everyone who has been addicted to this game and it's community over the past 18 years.

Author's note: My play time on Steam does not reflect my actual play time. There was an issue with the game keeping track of the play hours. This bug has since been fixed.

Author's note: I wrote a negative review about this game earlier. It actually compelled one of the owners to contact me so we could discuss some things. Because of this, some complaints were actually fixed or improved!


Do I recommend this game? That is a really tough call. As it stands, I VERY HESITANTLY recommend this game.


There is a lot to like about this game.

First, there is no shortage of items whatsoever. Many are quest items. Many can be equipped. Some are totally useless! In fact, there are so many items that it can be confusing!

Second, there are a fair number of classes to pick from and many items that each class can equip. Each class has its own perks and flaws.

Third, there are job classes a person can pick from. For example, I am a blacksmith.

Fourth, there are plenty of monsters to fight. Each one has its own sprite.

Fifth, there is a decent sized world to explore.

Sixth, there are a lot of quests in the game. Apparently there is an overarching quest in the game as well.

Seventh, the music for the game is pretty good.

Eighth, there are minigames as well!


However, there are some drawbacks with the game as well.

First, the game's store on their website is a tad distressing. There are items in the store that you can buy automatically. The good news about that though is that you can also get those items in game. There are really expensive "quick levels" that you can buy in the store as well. $100 will max your character's level and his skills. Although I understand that hosting a game server costs money, I think being able to buy levels is kind of crummy. One of the owners and I talked this over. We both understood each others opinion.

Second of all, the community really sucks. Think a bunch of 20 year olds acting like a bunch of 12 year olds. No maturity. No accountability. Nothing. A big part of this game is the pvp factor. With pvp comes drama. With drama comes name calling and swearing. They have a right to say those things, sure. This is not a children's game or community.

NOTE: There is a swear filter for the game, but is not working at the moment. One of the owners was alerted to this because of my other review and said it would be fixed.

Third, there isn't much direction in the game. I'm okay when a game doesn't hold my hand, but they ploped me into a world and I had no idea what I was doing. One of the owners and I talked it over and they are seeing what they can do to help newer players more. From my understanding, there is a larger quest that needs to be done. I think it's sort of like a puzzle where you need to go off to one place, get an item, take it to another person, who will give you an item, to take it to another person to give you another item, etc. However, after 10 hours of playing this game, I still have yet to find the starting point of it.

Fourth, people in this game are solid. You can't walk through them. So if someone wanted to block a path and not be afk, but fishing or something, they totally could. This can be useful for quests and pvp, but it could also be abused.


After speaking to one of the owners, I realized that they really do care about the community and new players. The idea for this game is cool. The execution of it needs some work. However, I think this game might really turn into something great.
