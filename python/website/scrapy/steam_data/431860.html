About This Game

Color Chemistry is an abstract color mixing puzzle game.

The game includes 42 colorful brain teasing puzzles!

The mechanics of the game is based on the modern theory of color. Additive and subtractive color combinations are used.

Your task is to solve a puzzle in optimal number of steps.

KEY FEATURES

  • Unique puzzle game mechanics
  • Peaceful puzzling atmosphere
  • 42 puzzles
  • 22 Steam Achievements
  • Trading cards
  • Color randomness of puzzle pieces
  • The game is for everyone - there is no age restriction content

System Requirements

    Minimum:
    • OS: Windows XP/Vista/7
    • Memory: 512 MB RAM
    • Graphics: 128Mb
    • DirectX: Version 9.0
    • Storage: 50 MB available space

First impression: concentration game

I have played 5/42 (?) levels. The premise of this game is to make all squares the same color. There are 6 colors, and you can change them in a circular fashion (both ways): Red-Yellow-Green-Cyan-Blue-Magenta-Red. The game adds difficulty by having a "time bomb" (10 second timer on a changing square) in some levels, and in some levels squares also add their colors to neighboring squares where indicated. (There may be other mechanics in later levels.) The puzzles are not really that hard, but they take concentration because if you mis-step, your score suffers.

I did not like that the game offers 6 changes on each color when only two of them work; and I missed keyboard shortcuts for the changes. Otherwise, the interface is doing its job well.

The game was not able to hold my interest for long because I found it more tedious than challenging.

The game runs well on my old Windows XP computer.

Posted: 27 February
The developer gave me a key to review this game and whatnot, yaddayadda.

This game is pretty damn hard. It looks really simple but you have to memorize which combination does what and then be able to do that in a timely manner for the timed levels and be able to see a couple of moves ahead to complete the levels with the arrows in them (from the trailer.)

Something that isn't in the game or at least not explained that well is how the combinations work. Here they are:

Starting with the block followed by what it is combined with then what that ends up making. Colors stay in their teirs (additive and subtractive) and do not combine with each other. It took me a while to figure that out to be quite honest and I think this will help you out if you're interested:

blue w/red=megenta
blue w/green=cyan

green w/blue=cyan
green w/red=yellow

red w/blue=magenta
red/wgreen=yellow

magenta w/cyan=blue
magenta w/yellow=red

cyan w/yellow=green
cyan w/magenta=blue

yellow w/cyan=green
yellow w/magenta=red

So basically you have different worlds that have a base block color that all the puzzles will require you to turn all the blocks in to. The first is red. You can see the first level in the trailer if you need more of a detailed explanation:

Click on a block.

6 colored wheel comes up that you can choose from (even though only two work)

Choosing a color from the wheel combines the two colors into whichever one I said it would above this.

Repeat until they're all the color you want.


Sounds simple in principle, but I did not find it to be. The difficulty spiked pretty fast. As another con, there is no settings button I could find. It's fullscreen only with the ability to silence the music or not and the ability to view the credits (do people do that?).

Honestly, this game seems pretty damn great and the only real reason it could be such a challenge is because the color combinations are not what I learned. I learned red, blue and yellow were the main three colors of the world, with blue and yellow making green instead of red and green making yellow. It's all pretty backwards to me and goes against what I was taught since I was a kid.

I'll update this if I find anything horribly wrong in the later part of the game but since all the reviews are hidden currently I figured I'd throw this one out there with such little playtime. This game seems to be worth $3, no doubt. However, as previously stated, I think this is a really challenging puzzle game and will definitely not be for everyone. Get this if you think your a puzzle god or if you want a challenge.
