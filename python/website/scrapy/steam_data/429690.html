29 March

Maps 'n Mods Update!



Three new maps and three new modifiers headline this update.

Here's the full change log:

- Added Dance level: four square barriers move in sync around the center.
- Added Loop level: a circular conveyor belt pushes everything around.
- Added Maze level: lots of destructible terrain!
- Added a Powerups modifier. Now powerups can be disabled.
- Added an Unlimited Dash modifier. When enabled dashing will no longer cost ammo, but the dash cooldown will be increased from 0.5 to 0.85 seconds.
- Added a Super Dash modifier.
- Can now cancel a Jouster boost (or the new Super Dash) by pressing the Powerup button
- Starting powerup modifier now shows which powerup is selected before battle. Useful for the "Random" starting powerup option.
- Fixed rare pilot "self-kill" bug
- Fixed Dash not working while turning with Keyboard controls
- Renamed game mode "Headhunters" to be "Ship Hunters"
- Renamed game mode "Pilot Execution" to be "Pilot Hunters"
- Default game mode is now Pilot Hunters
- Reorganized and renamed some of the modifiers in the Advanced Setup menu.

Hope you all dig the update!

About This Game



13 dynamic arenas provide a host of hiding places and hazards. Avoid turrets, death beams, and black holes or you're dead!

Run over pilots in Pilot Hunters mode, battle for the most kills in Ship Hunters, or join forces in Team Deathmatch!

Battle up to 4 players or add any combination of AI Ships.

Play with a keyboard or up to four game controllers!

Arm your ship with offensive and defensive powerups including missiles, lasers, jousters and shields.

Mix up play with Game Modifiers. Enable asteroids, starting powerups, unlimited dash, and so much more!

Public Review on behalf of the Opium Pulses Steam group

Disclaimer: The copy of this game was provided by the developer for review purposes

Few words about the game
Astro Duel is something that everyone among gaming community loves to see - new ideas created from something we know. Im literally amazed about how people can create something innovative, even basing on game like Asteroids. Game itself is awesome. I had to try it out against bots first and it was quite fun. Then I played it against my gf and she said that it was one of her best times we had together. Well then... ANYWAY, lets get to reviewing.

Advantages
  • Game offers you bunch of different arenas to play on, 3 match types and possibility to set your own few rules, so there is no way for you to get bored quickly
  • Nice, classic sounds. Brings back some good memories from childchood
  • Game was created to play with friends. And it worked out perfectly
  • Controls and steering is easy and nicely overlayed

Summary
As I couldn't think of any disadvantages I'll simply move right to summary. Well, game crashed on me 3 times in about 2.5h gameplay, but for unknown reasons, so I skipped that. Game provides with hours of engaging fun, even if you're playing alone, it still can be great against bots. Hope it will get some updates to expand its possibilities as there is alot of room to create new content. I also hope that one day I'll get to try it with 3 other people, it must be a crazy mess! So hesitate no more, just buy it, have fun and don't rage quit too quickly!

Personal score
9.5 / 10

El Jeffe
( 4.0 hrs on record )
Posted: 22 February
Fantastic game with local multiplayer focus, and some solid single player options for if you have no friends. The controls are tight on both keyboard and on game pad (I've only used wired connections for xbox one controllers for gamepad support initially, and it works great). Multiplayer is a complete blast, and is a good way to ruin friendships in a more skill based game than mario party 5000. It runs really well on the surface pro 3, so you can also take it to a (soon to be former) friend's house and humiliate them with your amazing piloting skills. In all seriousness though, it controls super tightly, and causes players to make decisions about ammo conservation, whether or not to rush to a powerup or try to snipe somebody trying to grab one. The game performs great on my surface, and works very well with my minidisplay to hdmi adaptor on my tv. If you have a liking of local vs games like street fighter or tower fall, and enjoy asteroids, or one of those two options separately, this is the game for you.
