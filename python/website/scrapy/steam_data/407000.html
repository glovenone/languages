Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We want to make sure that the final game will be fully polished and we need a community to improve the game, using player's feedback.”

Approximately how long will this game be in Early Access?

“6-8 months, with a full realease in 2016.”

How is the full version planned to differ from the Early Access version?

“More content, co-op, maybe new features(depends on community's feedback).”

What is the current state of the Early Access version?

“Pre-beta. Singleplayer, tutorial and almost all vital features are finished.
Current features:
  • Open island to explore.
  • Procedural objectives system, which provides infinite replaybility.
  • Jetpack!
  • Weapon upgrade system.
  • Automatic skills upgrade system.
  • Special Skills: Bullet Time, Invisibility and Nuclear Blast.
  • Vehicles!
  • Different zombie types.
  • Zombie bases upgrade themselfs.
”

Will the game be priced differently during and after Early Access?

“We plan to gradually raise the price as we ship new content and features.”

How are you planning on involving the Community in your development process?

“We want to use their feedback to polish the game and create the best game possible.”

About This Game

Operation Z is an open-world first person shooter. Kill zombies, destroy zombie bases, collect mutagen to upgrade weapons, complete procedural objectives and do everything you can to survive. Forces of the evil seized the island and you are the only hope for this world!

EARLY ACCESS FEATURES:

  • Open island to explore.
  • Procedural objectives system, which provides infinite replaybility.
  • Jetpack!
  • Weapon upgrade system.
  • Automatic skills upgrade system.
  • Special Skills: Bullet Time, Invisibility and Nuclear Blast.
  • Vehicles!
  • Different zombie types.
  • Zombie bases upgrade themselfs.

Road Map (In-development):

  • Co-op.
  • Steam Achievments.
  • French, German, Italian, Japanese, Korean, Russian, Spanish, Simplified Chinese, Traditional Chinese languages.
  • More content.
