5 July

List of recent updates

Howdy! I haven't announced the updates in a while, I've just been pushing them quietly. Time to fix that so that you have a better idea of what to expect :)

  • Bugfix: The buttons for the second-player in a Lobby had regressed to not be mouse-clickable anymore (hotkeys still worked). Thanks dani_as0 and bentley_maniac for the reports.
  • Fixed regression that was causing log-spam of loading SDF fonts.
  • Fixed it so the local player is always on bottom if there is only 1 local player (this was messed up in online games, previously... because we had assumed the local player was always Player 1).
  • Pressing "F" in Pass-n-play mode now works. This allows (for example) to play against yourself and swap the board each move so you always play from the bottom perspective, etc..
  • Fixed the occasional crash on Mac/Linux that occurred especially when entering the Online Game Menu screen very quickly after starting the program.
  • Fixed a recently-developed crash when opening the Steam Overlay (eg: for Leaderboard or Achievements) in OSX.
  • Fixed the case where you Leave a game during LoadingFromServer, then started another game (eg: local, or just a NEW online game) and the game thought it was still loading from server... forever.
  • Fixed game-reconnections (if it says "Loading from game server..." at any point during a game, now it should succeed again. That was broken for a while... we just didn't notice because that only happens in weird cases anyway).
  • Fixed the use-case where a host starts a game asynchronously with the opponent moving first, then a guest joins and makes a play, then the host would previously freeze in "Loading game from server...". Creating async games and having the opponent move should be fine now.
  • Rewrote IAP to use DLC instead, which will eventually let people get card-drops, allows the game to show up in Steam Libraries, etc. This is a really complicated issue, but we think we're very close & we just need Valve to change one config setting for our app and we'll be all set.
  • Added Condal Chess Set 2D skin to Premium version.
  • Game warned "this game is rated" when forfeiting, even if it was not rated.
  • Fixed a collection of crashes on OS X. Found the magical version of Mono that we needed (it turns out that a rather old version of Mono seems to work the most consistently across all common versions of OS X). This was fairly urgent for Mac users so we just pushed it out the moment we found out it was working for all testers. That's one of the reasons that I didn't make a single clean "release" with an announcement. Sorry for the delayed announcement! ^_^;

We have several other things in the works including translation, one unreleased fix (it says "Local Multiplayer" on the top of the "Online Multiplayer" lobby), etc.. Stay tuned :)

Oh yeah, here's where you can get the Premium version (which isn't as useful until we get those card-drops working, but we appreciate the support if you want to grab it now, anyway!):
http://store.steampowered.com/app/451690/

About This Game

The classic board game comes to Steam! Everything you need in a chess game. If you are a chess enthusiast or someone who just likes to play a game now and again, this is what you've been looking for. This is Simply Chess.

Play it against friends whether they're on on Windows, Mac, & Linux.

Features:
  • Play locally or online.
  • Play against the computer â World class "Stockfish" AI with 100 difficulty levels.
  • Includes "hot-seat"/"pass-n-play" mode for multiplayer using only one computer.
  • Asynchronous play - you can play even when your opponent is offline.
  • Easily switch between full 3D with camera-control & a 2D graphics mode
  • Full controller and âSteam Big Pictureâ support
  • Switch seamlessly between playing with mouse/keyboard and gamepad.
  • Elo Ratings for online games â ranked on a Steam Leaderboard
  • Steam Trading Cards, Badges, Backgrounds & Emoticons
  • Dozens of Steam Stats & Achievements.
  • Steam version has no added DRM and can run everything except the online play & Steam-specific bonuses (Steam Achievements, Steam Leaderboards, etc.), without any internet connection and/or without Steam installed.

Simply chess is a simple and plain software that is not demanding on the machine.

BUT just because it is simple, does not mean it doesnt need improvment. Because it does.

1. There should be options to change the sound effects when moving the pieces.

2. There should also be more color selections/themes for the boards. Tournaments are almost always done on a green and white board. The green and white chess board is one of the most popular chess boards and needs to be a option in this game.

3. after beating the computer on a certain level, there should be a button that takes to the next level without having to return to the main menu and selecting the next level. Also, if you go back to the main menu and pick an AI game, the software should automatically be setup for the next level if you have beaten the previous one, instead of having to click the next level.

4. The control keys for the game should not be listed in the "how to play" section of the game. The keyboard controls should be listed in the settings menu.

5. The "how to play" section fails to mention that the right click on the mouse is used for canceling the move.

6. ADD the option to add coordinates on the board. File and Rank coordinates (a - h & 1 - 8).


All in all, this is a simple program which has a lot of potential if simple fixes were installed. Lastly, the background classical music is awesome and puts you in the mood for chess. :)

This game is what you would expect from a Chess game; it's good old Chess on Steam, and it's free.
