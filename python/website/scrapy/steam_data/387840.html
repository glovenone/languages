About This Game

Experience a challenging and addicting platform game, that never ends (until you die). Alpha Runner keeps players on their toe's by creating different scenerio's so the game is not quite as easy as it looks. Alpha Runner has both Global and Friends Only leaderboards which allow you to be the very best between friends and others. With Alpha Runner's randomly generated worlds, you cannot plan how it will play out. Alpha Runner is a must play for anyone who wants an extreme and competitve challenge.



Alpha Runner will have frequent updates which contain new obstacles and game-modes to make the game even more enjoyable.

Posted: 6 September, 2015
Summary:
Alpha Runner presents a fun, original concept which includes randomly generated levels, several different character types, mini games, online leader boards and different game modes. The game can be buggy at times. The achievements are extraordinarily easy or 'grindy'. However, the worst thing is that your character is randomly selected for you at the start of each run, instead of you choosing it. In its current state, the game itself is quite fun, it just needs some real achievements, bug fixes, proper options and a character selection menu. I personally think you should wait until it gets out of early access, by then it should be quite a nice and complete game.
The Good:
  • Online Leaderboards.
  • Fun concept + gameplay. Basically, you jump platform to platform, trying to dodge traps and collect score. The platforms will slowly disappear below you, so you are required to keep moving. There will occasionally be mini games to play.
  • Randomly generated. The levels and are randomly generated and have some fun mini games to play in them.
  • Different characters. There are several different characters with different abilities: Space Jump (higher jump + floating effect), Double Jump, Gotta Go Fast (run faster), and Magnet (attract the score).
  • Music. There is only one short, looped track (for now?), but I find it very entertaining to play to and it sounds quite nice.
The Bad:
  • No resolution options.
  • Can't remap controller controls.
  • Random FPS drops. For the most part, this game runs very smoothly. However, it occasionally randomly drops to about 35- 47 FPS and stays there for several minutes. Currently the game is basically unplayable for me because of this.
  • No character selection. At the start of a new run, the game will randomly select your character. It even does this for the speed run mode (which is peculiar since some characters affect speed). Different characters can make the game much, much easier, resulting in higher leader board positions. Having separate leader boards for each of the characters would be nice. Having to kill yourself until you get the character you want is just annoying.
  • Easy achievements. Usually achievements are big plus for me in games, however in Alpha Runner they fall into two categories: extremely easy, requires grinding.
  • Bugs. Including, but not limited to: sometimes when you fall below a platform, you will teleport back onto it (when moving or jumping semi-into it), pausing the game with a controller doesn't properly work.
  • Overlapping text. Ever so often there will be some ugly overlapping text, usually at the start of a run.
