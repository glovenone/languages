About This Game


You are a pilot in the government forces stationed at one of the planets under attack. As you get higher on your career ladder, you will be able to choose more advanced hovercrafts and ammunition, as well as to order your own squad and influence local battles.

The goal of the development of the game is to further expand our virtual universe. Based on the huge success of the first game, we are presenting you with the same world, occurring at the same time, but from the point of view of another character, whose adventures unfold according to a different storyline. At times, it will cross the storyline of the previous game, and if you have played "Echelon", you will recognize some places and characters.

Echelon: Wind Warriors is a unique game. This is not a regular space-battle simulator, nor is it a simulator of combat in enclosed labyrinths. The action is taking place above the surfaces of a planet, the picturesque landscapes of which are one of the basic features of the game play. In the realism of the implementation of the gaming environment â the cockpit of the battle hovercraft â Echelon is comparable with the best flight simulators. The game play is similar to that of the MechWarrior series, but instead of a walking robot, you pilot a fighter. Echelon: Wind Warriors has the following advantages over Echelon:


  • Unique combination of flight simulation and action.
  • Realistic and highly detailed world of the game. Various, elaborate, and vast landscapes.
  • A broad but well-balanced range of available weapons provides the player enough freedom and flexibility for using different tactics.
  • The playerâs progress, as he advances through the game, upgraded equipment, as well as the appearance of new equipment with unknown functions on the part of the enemies will continually add excitement to the game play.

I bought this because it's a classic that I spent a lot of time on when I was younger. It's a fun flight sim with a lot of missions to do.
- Some people might not be able to run this game properly on modern machines, but that's fixable with a little bit of registry editing, setting the correct settings (Like choosing Hardware TnL in options app), and downloading some extra missing files (link in comments).
- I highly recommend setting the control method to "E-mouse", with inverted Y-axis (Makes it behave like an FPS).

If anybody has any questions about how to get this game running, comment on my review your questions, and I might be able to help you with it. I'm running this on Windows 8.1, with x8 Anti aliasing at 1280x720@60 FPS and all settings maxed (Trying to figure out how to get 1080p, at the moment).

TL;DR If you ignore the outdated visuals and the hassle to get it properly set up the first time, this would be a fun flight sim for you.

There I want to tell a lot about this great game.
Basically this is standalone addon for original Echelon game a futurstic action/VTOL-simulation set in same universe and period but from another character perspective. The game features decent flight model which makes you feel of flight and inertion, advanced damage model, a lot of detailed ground, naval and air units and buildings with their own damage models. There many types of aicrafts and weapons to choose. The game is unique by being realistic at some point Sci-Fi vtol sim. The game locations is large, the whole continent is featured here.
The story/campaign is good, long and even non linear at some places, the fully and nicely voiced dialogues (yeah with yelling commander etc.) also.
The graphics of original Echelon (2001) was next gen for it's time, but in WindWarriors not really much changes.
Game mechanics have a lot similarities to MechWarrior series just you control the aircraft there.
Sadly this game didn't got any sequels (forget about operation matriarchy) so the story is incomplete due publishers counted this type of games as not profitable.
There is campaign coop and nice multiplayer with gamemodes involving teamwork and just fun deathmach, the bad thing it was using Gamespy. (the russian comminity still run server, but it reqiure custom patches) Unlike most of harcore sim games this is easy to learn but hard to master, in multiplayer (really!). There more action than fight with controls of your machine.
I would recommend this game if you like open locations, Sci-Fi VTOL, atmoshpere (in both meanings, game take place only on planet) some hard missions to challenge your skills, epic and dynamic battles and unexpected turns of a plot.
