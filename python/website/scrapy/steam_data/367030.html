Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“Many of my fans has requested it, and I would love to have them as testers along the ride. Having people playing my game during development will be a great motivator to work harder :)
I have saved up for an opportunity like this, so money is no issue.”

Approximately how long will this game be in Early Access?

“Planned release is early 2017, about one year left.”

How is the full version planned to differ from the Early Access version?

“My end goal is to have about 6hours of varied gameplay, it currently has about 1hour of somewhat repetetive gameplay.
Planned features are:
-Collectable monsters
-Varied levels with some puzzle elements.
-Interactive environment, like throwable barrels and levers.
-Online co-op.
-Multiple stage bosses.

Players in early access will treated as beta testers, and will have access to debugging tools to mess around in the game with, can use in-game editors and will also be able to test all secret levels and weapons before they are hidden. These features will be removed in full release.”

What is the current state of the Early Access version?

“There are 15 levels with bosses, a total of 1-2 hour of gameplay. The game is fully playable but just about every aspect of the game needs to be polished. Can be played with Keyboard, Mouse & Keyboard or Xbox360 controller.”

Will the game be priced differently during and after Early Access?

“The price of the release build is planned to be $3.”

How are you planning on involving the Community in your development process?

“They can help me with both playtesting and feature suggestions. They have access to the in-game editor and can suggest new designs on items, characters and terrain.”

About This Game

Jump in to a party-style adventure game, bash down monsters within a minute, grab some loot and save the day!
Lootfest is an action adventure game, in 3rd or 1st person view. Random generated voxel world with a fantasy setting.

Current early access features:
-Contains 15 levels with monsters and bosses.
-Voxel Editor, change the terrain anywhere.
-Model editor, modify any in game model.
-7 weapon classes.
-You got access to all debug tools in the game, for easier playtesting and bug reporting.

Technical Facts
Input: Keyboard, Mouse & Keyboard or any controller
Big picture mode: Yes
Rebindable keys: Yes, for any input device.
Multiplayer limit: 4 local splitscreen, 80 online
Supported resolutions: All
Multiple monitors: Yes

Max Field of View: 120
Anti aliasing: No
SSAO: No
Offline gameplay: yes

Lootfest is fun for a short period and very pleasant, but it is also quite unfinished. You're unlikely to find a multiplayer match, unless you've got a buddy to play with. Early Access Review
Posted: 18 May, 2015
[Updated for current state of the game 10/8/2015]
Okay so I'm marking this negative specifically for the current state of the game.

Currently the game offers very little for people looking for a game to waste a decent chunk of time in.

The class system is setup where you just pick up 1 of 8 current weapon types that allow you a basic attack and secondary.

There is currently no character progression system nor equipment system (ie: there is no actual loot in the game currently)

to break it down a bit:

Currently lootfest doesn't have a plot or point there is no story (early access so I can accept this)

The control is similar to most voxel hack n slash with the exception that due to the map style the camera can be wonky

The feel is non-existent, you are random character A that randomly kills monsters with which ever left or right click attack you want to save random baby A over and over

The play-ability is one time. Until decent changes i don't feel the need to play it again because no matter how much I play it I will always be random character A equipped with random 2 attacks (A B C or D) that walks through a map to collect keys and for some reason kill a boss for a baby. I wont ever become more than that in the current game system and some random character B that just bought the game could walk in and be the exact same as me despite random character A spending way more time in game playing.

You have the ability to make your own stuff through their voxel system which is a good thing, I've seen it be very successful in other games but this one is still very early in its creation so the creating system is not at a point where it can offer much.

The available maps to play are very straight forward and due to no equipment or leveling there is no point in killing anything on the maps except the bosses (which currently drop a random baby followed by you being teleported back to the starter map)

Now I do expect the game to get much better as the devs work out where they're going with it.

If you feel like helping them move along by buying it in early access don't expect to get a lot of enjoyment out of the current version of the game but rather probably bug test the current version a bit and hold onto your early access version until the final version is complete because it should become a pretty good game.

Hope this helps :)

[Update]
I updated the review after I replayed the game, all I needed to change in the review though was the amount of "classes" from 4 to 8. Nothing else to note has really changed, I went through all current levels in ~15 minutes, still no point to killing anything but the boss, still no real progression or something to work towards, and no loot so I'm still gonna be waiting until it's a good game. Leave me another message when the game has made decent progress or if you want me to re-evaluate the game to update my review.
