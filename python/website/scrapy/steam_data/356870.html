Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“I chose early access so I could get my game out to the world and start getting feedback. If it were just a standard RPG I would not bother but since Devoured Time will have much more I decided to go with early access. The game is still very early in development so I can take feedback and tweak and/or add completely new features to the game.”

Approximately how long will this game be in Early Access?

“That depends on how much feedback I receive. If I don't add anymore systems and features i would estimate the game itself will take at least a year from launch to complete, but that's just an estimate.”

How is the full version planned to differ from the Early Access version?

“Early access falls between alpha and beta and will be missing much of the main story. The first few months will be more or less a systems test and implementation. During this stage I will tweak and add new systems based on feedback. The final game will have the full story and all the systems I mentioned on the main page.”

What is the current state of the Early Access version?

“The current state is between alpha and beta. The first few version that will be playable are a few behind the main game and are for testing purposes.”

Will the game be priced differently during and after Early Access?

“Yes, since the game will have limited content it will be priced much lower during early access and subsequent releases will see the price increase based on content.”

How are you planning on involving the Community in your development process?

“Since the game is between alpha and beta there is a lot of room to expand so I want the community to throw ideas they want and if I can I'll implement them in. Besides the main story I'm open to any feature that will make the game more enjoyable. Plus bug feedback will help to.”

Progress Update

Another update without a video, but I think next update I should be able to create one. By then I will have a ton to show. But like before much has been done since the last update so here goes.

Enemies

All enemies are in, animated, and take and deal damage. Overall there are 15 different types of enemies, and few sub types within each class. Each enemy will share some weaknesses with other enemies, some have more than one. More on enemies in the next video update.

NPC’s

All key npc’s are in and have their first stage dialogue in.

Tools

Each of the 4 tools are in and each function. More in the next video.

Current Task

Currently the top priority is finish the Rock/paper/scissor content. After that’s in its really just cluttering the map, tweaking quest and bug test/fix.

Time Frame

It’s looking really good to have the game complete by the end of this month. Then I plan on taking July to polish the map, animations, etc while testing for bugs and releasing the game in September. Keep in mind while the game will be released I will still be adding content.

About This Game

Early Access (Current Build)

The early access build features a 20 + quest line spanning 7 different zones. Minimal inventory management, 4 skills, 2 weapons, and one playable character are the basics of this build. This build was used to establish different systems along with game flow mechanics which are being implemented into the final build.

Final Build (Releasing by End of 2016)

Experience a unique story following one of two playable characters. Discover your past; learn about the world through lore books and lore trinkets. Find special weapons and upgrade them to fit your combat style. Manage your home base and seek out others to join your cause. Hire mercenaries and take them into your strategic battles. Explore and find secret treasures along with hidden collectibles. Each one of the 10 zones are crafted from scratch and made to have their own theme.

Check out announcements for links to video updates with final build info.

Early Access Review 08.05.2016

Devoured Time is a RPG / City-Building / Adventure. Can i call it so ?

Either you start the Story which is currently 11 Quests or you start the Survival Mode that is still in work.
In the Story Mode you gather resources, rebuild the City, craft Weapons and other stuff. We also try to find allies and educate Soldiers to protect our City.

While the Buildings we create look pretty good, the rest of the environment looks a bit "mushy". The Gather/Craftsystem needs also a bit work, what has already been announced by the last updates.
The Day and Night change and Weather also look good, for the Night, a craftable Torch would not be bad :D
So far there are not many opponents or other Cities. After 3-5 hours you have seen all what is now available.

The Developer seems to be quite friendly, I got quite fast a response as I had a problem.
If you wanna see what coming next, on youtube you can find some videos.

Early Access Review
Posted: 29 October, 2015
I really don't want to give any game a negative review but since Devoured Time isn't a game yet I have no problems this time. I bought the game, started it up, and found tabs for a handful of different game modes. Alas, only the tutorial and the main game tabs seemed accessable so I chose the tutorial. Once in I was unable to interact with anything at all, thus rendering the tutorial a waste of time. Okay, that's fine, it's EA of course, so I tried the main game. That turned out to be the exact same thing as the tutorial. The only other playing options were 2 tabs that were for testing- TESTING!!

I decided to withold any negative thoughts and instead posted my problem on the forums, thinking maybe I had missed something. The next day one of the Devs (beast) replied to my post stating I shouldn't have been able to access the main menu tab in the first place because he is still working on it to get it ready. Oh really? Then why is this product ( I refuse to call it a game anymore) on here? I can't say for sure but with such decietfully misleading info being given to entice the unwary to purchase this product, I don't have much faith in the people involved as of now. I may be shown to be wrong and hope I end up wrong. If that happens I will personally apoligise to Beast and will update this review to reflect any positive changes taking this product foreward but for now stay away from it. Star far away!
