About This Game

Travel back to 90s' games with the Ninja, a mysterious killer whose true purpose nobody knows, and live a full retro gaming experience in a world totally drawn in pixel-art.
Fight against mythological monsters, aliens, robots, gather coins, collect all the potato chips you can and see a surprising uncommon ending.

Features

  • Classic game play
  • Up to 3 different Sub-Weapons
  • Super ultra original "Up+Attack buttons" system to use these sub-weapons
  • Hidden paths through stages
  • 3 alternate endings
  • Unlock a hidden character with unique abilities
  • Fight a lot of unlikely enemies
  • Humorous monster database

Well, this game is bad just a bit too much. First level is fine, boss battle is fun, but in later levels things go down the toilet very fast.

3.9/10 - Retro, Amateur Pixel Platformer

Price is a relevant factor when doing my reviews. $5 for this must be some kind of joke; the acceptable price for a rough pixel game such as this is approximately $0-1. Nothing new invented here; there are far better and cheaper platformers out there.

My main issue was with the controls, they were imprecise to say the least. If you want to jump to the next platform you have to really exaggerate the motion to successfully make it. There is no room for quick reactions that can save you from dying. I just like to have complete control of my character; losing because the controls are wonky is an automatic âthumbs downâ.

I know the sequel has been Greenlit... but how? and why? Worth $0.99
