About This Game

Stoorm is a skill arcade game. Although the reflex is your main asset to overcome all levels, you will sometimes need to take back your breath, and a minimal analyze of the map, to escape without dying too often. The purpose of the game is to move your cube through the rooms to finally leave the complex or... you 'll see. Stoorm is more than 200 rooms. Simple gameplay. A colorful design, pixelated and dynamic. Music by Zentoy, courtesy of Rainbow Mode Records. The game is played with the joystick and keyboard. Stoorm Yellow version have new enemy and player alternative. You need lot of skills to end the 60 levels of this version.

First impression: cheap timing game.

It's all about avoiding moving objects. Well, it would be if it wasn't also dealing with annoying scrolling backgrounds (seizure danger?), an avatar moving just fast enough to run into any bot you're following and die, and the angular avatar means that turning a corner is a bit like threading a needle.

There's no way to exit the game, "partial controller support" means keyboard only (as far as I can tell), the "play" button on Steam always runs the installer, and the game's native resolution is 965*620 or thereabouts.

Not my cup of tea, and even if it was, I'd probably think it slightly overpriced.

Crashes Steam if you happen to press the play button.

Workaround: go to the game's folder, extract STOORM-Game.exe with your favorite archiver (it is actually a self-contained archive) and launch the game from the resulting directory.
If you want to be able to launch it in Steam, you have to move files back in the original directory and rename STOORM.exe to STOORM-Game.exe .
The game itself is an utterly basic action game. Nothing to write about.
