About This Game

Do you love puzzles with dropping, pushing and connecting artifacts ?
Then welcome to our matching artifacts game with brains ( brain not included, you’ll have to use yours )

Help Dale Hardshovel the famous archeologist solve the Bloomstone Mystery in this fast-paced and exciting Puzzle game! Can you find all the pieces of this ancient artifact and solve the timeless mystery?

Features


  • 100 Pure Mind-twisting Levels
  • Unique matching artifact gameplay, with various helper elements like moving platforms, teleports, magnet and switch, building blocks
  • Story driven quest
  • Exotic trophies for outstanding performance
  • Original in-house composed soundtracks that revive the spirit of ancient Egypt
  • Future updates will bring in-game level editor for custom created levels shared among players

Dale Hardshovel HD is the remastered edition of Dale Hardshovel, mostly the graphics and the additional worlds are different. The levels didn't changed.

This game is a matching game, the aim is to clear all items of a level by matching twins or trios of the same item in a limited amount of time. It remember me this kind of game, he it's not that one, on the first mobile phone with a color screen.

The mechanisms of the game are easy to learn and master, some levels are really challenging. There are also some environments to be aware of or to use, such as: weak blocks, sticky black oil, moving spikes, portals, magnet...

(*):
There is only one main glitch that is required in order to finish some levels. In one level you need to pause the game while using a moving platform to save time. Without doing that, you are always running out of time.

This is still a great game to play, and I recommend to use my guide if needed to finish the game or getting the achievements.

Dale Hardshovel HD is a good puzzle game. The controls are simple: you move the artifacts horizontaly with the mouse. You need to match the artifacts, in pairs of 2 or 3 artifacts, in the given time to be able to complete the level.

A little more info about matching: for the matching to count they have to be next to each other in a "stay" position...if you swipe an artifact over more artifacts and there is one of same model under it won't count as a match. Also if an artifact is on the moving platform and another of same time on the ledge near the moving platform when they will be next to each other it won't count as match.

The complexity of levels differ based on the things you'll find in them: from movable blocks that will help you to get over holes, spikes that will destroy your artifacts, blocks which dissapear after you passed them a number of times to teleporters, moving platforms(which can't keep more than one artifact at a time), magnets and switches.

As a side note, for long play sessions the soundtrack may become a bit annoying because it gets repetitive.
