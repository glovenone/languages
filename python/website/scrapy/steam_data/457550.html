Alpha Branch

To test early features we have a public alpha branch on Steam. This branch will always be compatible with the main branch. We really appreciate your feedback and please let us know when we break things!

To join the alpha you'll have to exit BigScreen, right click on BigScreen in Steam, go to properties, betas, then select alpha from the dropdown.

The current list of different features / known issues in the alpha branch will be updated here: https://steamcommunity.com/app/457550/discussions/0/351660338715016537/

22 June

BigScreen added Oculus Remote support to control your entire desktop

Hey everyone!

BigScreen 0.5.1 is live on Steam

Changes:

  • Left / Right changes screen size
  • Up / Down changes screen distance
  • Hold back button to move screen
  • Press select while looking at your screen to control your mouse by gaze
  • Press select again to left click
  • Press back button to stop controlling the mouse

Let us know if you have any problems with this control scheme and if you find it useful!

About This Game

With BigScreen, you can have a Virtual Reality LAN party.

Use your Windows desktop on massive screens in a virtual space where you can hangout with friends, watch movies, and even play video games together.

Features

  • Play your favorite games, including games on Steam, on massive screens in VR.
  • Watch anything you can see on your desktop screen - Twitch, Netflix, YouTube, even VLC - on IMAX-size screens
  • Choose and customize your screens, from small 30-inch screens to curved 65-inch TVs or wall-sized 20-foot displays.
  • Adjust your screens to the perfect height, position and curvature for ergonomic comfort.
  • Choose environments from outer space to a luxury apartment
  • SBS and OU video support (works in multiplayer too)
  • Vive controller support
  • Oculus Remote support

Multiplayer Features

  • Private or Public rooms for multiplayer, supports 1-4 players
  • Cross-platform support for Oculus Rift and HTC Vive
  • Low-latency positional audio chat
  • Private & encrypted screensharing in VR
  • Leap Motion support for VR high fives!

Coming Soon

  • Shared desktop audio
  • Customizable avatars
  • More environments
  • Better performance

System Requirements

    Minimum:
    • OS: Windows 8/8.1 or Windows 10
    • Processor: Intel i5-4590
    • Memory: 4 GB RAM
    • Graphics: NVIDIA GTX 970 / AMD 290 equivalent or greater
    • DirectX: Version 11
    • Network: Broadband Internet connection
    • Storage: 800 MB available space
    • Additional Notes: Windows 7 is not supported
