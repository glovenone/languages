It is true that this game can be pay to win, but it is still actually a very fun game. Treat this game as the typical "lite" version unless you are willing to put money into it. That being said, it is a BIG lite version.

Pros:
Dungeon Crawler and builder all in one game
You make and customize your own castle and dungeon with several types of traps and monsters
Lots of loot
Constantly upgrading gear
Easy to learn
Not very buggy unless you are very unlucky

Cons:
Only four character classes
Only get to choose one class (unless you pay real money or work your tail off)
Vast pay to win system
Highly competitive and repetitive
Can get boring
Limited number of abilities/spells and an even more limited action bar (3-4 abilities on your rotation, really?)
Slower leveling than most games

Generally, if you treat it as a "lite" game and want something casual, this game can be a lot of fun. If you want to succeed past casual, though, you pretty much HAVE TO DUMP A LOT OF MONEY into it.
