About This Game

Disney Infinity 3.0 Edition now welcomes Star Wars™ to the ever-growing collection of Marvel, Disney and Disney•Pixar characters, stories and worlds. Together new heroes can join forces with characters from previous editions and embark on adventures as big as your imagination in the 3.0 Toy Box! Disney Infinity encourages you to play your way, whether you explore open-world creation in the Toy Box or story-driven gameplay inside Play Sets!

There are plenty of accurate reviews out there, so I won't repeat all of the information others have already provided, aside of believe that this game gets to be very expensive, very quickly. Overall, I would say pass on the PC versions and go for a console version. If you do not have a console, I recommend buying the character models on sale somewhere and using the code card to access the characters in the game. That way you do have some resale value and you ultimately will still be able to play the PC characters.

I bought the game, hoping for better graphics than the PS4 version could provide, but ran into the following issues:
- there is no way to use the save from the PS4 version on PC. Disney needs to implement Cloud Saves to have progress portable between any system.
-Screen tearing. There are minimal graphics options for this game. Overall graphics are not much improved beyond frame rate.
- No multiplayer. The console versions of the game had local multiplayer. This shortfall means that my son and I cannot game together like on the console version.
