About This Game

"Dreaming" is a short episodic minimalistic first person story, of a man's journey to the Dreamingland. He is determined to find the girl who calls to him. Are you ready to go through the horizon and find the impossible world, the place where reality and dreams are intertwined?
  • Built with CryEngine.
  • Feel the harmony of dreams as you venture into the surreal.
  • Crisp low-poly style ;)
Dreaming was inspired by ThatGameCompany's masterpieces.

System Requirements

    Minimum:
    • OS: Windows 7 64-Bit, Windows 8 64-Bit
    • Processor: Intel Core 2 Duo 2GHz / AMD Athlon 64 X2 2GHz (multi-core processor is strongly recommended)
    • Memory: 4 GB RAM
    • Graphics: NVIDIA series 400 GTX / Radeon HD 6000 Series
    • DirectX: Version 11
    • Storage: 1 GB available space
    Recommended:
    • OS: Windows 7 64-Bit, Windows 8 64-Bit
