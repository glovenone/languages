Version 3.3 Released

We just released version 3.3.0 of DP Animation Maker, which comes with some minor bug fixes, but also some new features.

  • New Through space animation
  • Improved the appearance of the Rings on water effect with small backgrounds
  • Enabled the camera in the Rings on water effect
  • Fixed the missing pedestal picture in the Glass ball effect
  • Reviewed the Import animation options, improving its performance, memory usage, and quality of the final picture.
  • A new Clear edge option in the import dialog
  • Added a spin option for the Static imported animation
  • The path animation now starts from the first point on each export
  • Added a spin option for the Static imported animation
  • Fixed occasional crash on opening the Add animation dialog
  • Fixed a problem with loading the animation library on several computers
  • Projects with the manual time line now save correctly
  • Pressing the Enter and Esc buttons does not hide the slide property area anymore

About This Software

DP Animation Maker aims to provide non-professionals with powerful animation tools that turn almost any 2D art into stunning animation. With this program, you can make the following with minimum effort:

  • Animated footages and backgrounds
  • Animated introductions for games
  • Animated pictures for multimedia products

The software toolset includes the following powerful instruments:

Smart Animation Brushes

Smart Animation Brushes is a unique feature of DP Animation Maker. It is allows you to paint over a still image turning a picture of water, fire, or light into real-time animation. Creating professional-looking effects have never been so simple.

Morphing/Deformation Brushes

These brushes give you a simple but effective method for bringing life into static pictures. Make stars spin, plants wave, ships sail and many other cool movements with little effort.

Reach Library of Ready-to-Use Animations

Several hundreds of ready-to-use objects and elements are available in DP Animation Maker and its four add-ons. They include fish, butterflies, sparkles, flowers, grass, sea-plants, and many more.

Automatic Looped Video Generation

DP Animation Maker supports six output formats with an option that allows you to create seamless animations automatically. With this smart feature you can create a seamless video of any length.

    • Memory: 1 GB RAM

( 30.0 hrs on record )
Posted: 8 March, 2015
DP Animation maker is one of the best 2D/3D applications I have used on the market. The program comes equipped with many background templates that are usable, customizable and exportable. There are also a variety of options to use to add realism, color and depth to your animated scenes. I especially like the fact that you change backgrounds, using your own JPEGs, PNGs, as well as TGA and Bitmap files and you choose from a variety of special effects to add to backgrounds, since as a graphics artist, I need to use effects such as water, fire and underwater effects. I really enjoy using the "no background" feature as well, because it allows you to export single or multi-object animations that can simply be exported with transparency, for example, when exporting a sequence of transparent PNG images or as a video file. This exportation feature with transparency is particularly great for gamers and game developers who want to be able to extract content and then import or drag-and drop into other applications with ease.
I love the fact that you can also export many royalty-free animations and use these to either create other scenes or use them as standalones in a variety video formats that DP Animation Maker provides. As I tend to make scenes with characters and I do a lot of promotional videos for clients, I am glad to have animation options, such as; fireworks, clouds, light rays, lightning, snow, stars, smoke, as well as certain animals such as birds, and many tropical fish that are already part of this excellent program. Noteworthy, is the fact that you can set different timings for animated objects within a scene; set direction, motion and speed, as well as flip objects and set the number of times an object occurs as well as how many duplicates can be made of objects you add or import into a scene or on the blank canvass. I am also happy with being able to import my own sound files into the program and set realistic camera movements from the options available within the camera, such as; rotation, speed and zooming.
Above all, this is undoubtedly an excellent product and I wholeheartedly recommend it. Besides, you get good value for money and you also get the options of adding your own animated objects to scenes and you can buy other packages of animations from the developers' website to add more realism and moving content. Moreover, the customer service is exceptional from the developer’s website, as I have been able to get responses from their technical support staff within 24 hours of sending a support e-mail. Keep up the good work guys at http://www.desktoppaints.com I recommend the product to anyone and everyone who creates games, websites, logos, videos; designs or produces animated content and just about anyone who enjoys making animated scenery for personal or commercial projects.

I stumbled across this utility on steam while looking for a way to animate business logos / banners for websites and personal use, I’m glad that I did; it has definitely helped me bring a few images to life as well as presented itself as a utility to make custom graphics for presentations. The animations that this software can put out are a perfect fit for powerpoint presentations, which have helped add some eye candy to a few slide decks I’ve been working on. I am not a professional animator by any means, but I found this software very easy to use; I was able to pick this tool up and quickly put it to good use; I also found the developer’s introduction video and the developer’s website manual as a great help.

I really like the range of available animations; while I think that some the animal animations aren’t very lifelike they definitely add some novel ambiance to the images that I’ve made and the logos that I’ve created; a random eagle flying across a Made in America logo captures the eye well.

The software operation has been solid, I have not had any crashing or issues with either my windows 7 or windows 8 machine. The developer states that the purchase price includes lifetime updates, which is a huge plus because I hope they keep adding content. I am really looking forward to what new animations and abilities are added to this software as I’ve had a good time just playing with the animations that are available. Personally, I would like to see more variety added in the types of animal animations that are added as well as the types of brushes. I think this software has a lot of potential to help amateurs like me as long as it keeps expanding.

If you are in the market for an easy to use animation tool, I recommend this product. I hope that me taking the time to review this product helps others make the decision to give it a try.
