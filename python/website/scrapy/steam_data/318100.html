Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We need your feedback of the game to improve all gameplay aspects by the time of full release. Besides, some features like WARGAME mode weren't implemented yet. Nevertheless, you can already feel that such the MOBA concept inside of a multiplayer shooter!”

Approximately how long will this game be in Early Access?

“Approximate date of full release isn't clear for us but we will try to finish the game as soon as posible. Game will be in Early Access until it moves from Alpha to Release version. By then, we realize all scheduled features.”

How is the full version planned to differ from the Early Access version?

“In the release version, there will be added new achievements, new weapons, a variety of unique characters (heroes) and new maps. Animation quality also will be improved. In addition, the full version is very different in terms of user interface. For example, there will be added UI for WARGAME mode.”

What is the current state of the Early Access version?

“Now game is ready as a multiplayer shooter with the MOBA elements. It's possible to tell that game is on alpha version stage. It's does not mean that you should be afraid of bugs. We are well tested all functional before the game to release.”

Will the game be priced differently during and after Early Access?

“Yes. Currently, as game isn't finished yet, the price is reduced unlike release version.”

How are you planning on involving the Community in your development process?

“We would like to hear your opinion on this game concept at the forums. And also, with your help, it is ideal to tweak balance of game and to make it the most convenient!”

About This Game

AXYOS is an experimental FPS being made for players by players!

Inspired by giants of gaming industry, we made a decision to create our own game, which would inherit some key aspects of FPS and MOBA genres but was built relying on opinions of common gamers. This is how AXYOS came to be. Together, we can make this game happen!



To say it simple - this game is a combination of a MOBA genre and mostly multiplayer FPS. We are trying to implement MOBA elements such as Ingame shop and session level-up system inside of a shooter mechanics. By the time of full release, we will introduce more and more of such elements and balance these two genres within one game.

We are set to leave the Early Access program and get to full release as soon as it is possible!
