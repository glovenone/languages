I started playing this game,and I got confused the moment the interface opened. The first thing I did was start the campaign. A good thing because it started learning this game. The problem is it takes a long time to explain what everything means. NUM: n to n, SUM: n to n, sum %2. Even after playing through it all, and eventually finding out what it means. A few moments later, oh yeah, what does this mean again. No tips if you hover above the text. Try to find the right puzzle to find the description again.

This doesn't mean I don't enjoy this game. I like the concept and the puzzles. Logical thinking and math. I say yes to that. You can get into the game. It's an easy concept but addictive. It's a gem but unpolished. And therefore I do not recommend it (yet)

The came should be polished, and a lot.
cons:
Timer doesn't stop when finished.
Descriptions and interface confusing. Need more tooltips.
Hints show immediately. It ruins the fun in a logic game.
When finished a puzzle in campaign, no continue in screen, but on top of the screen there is a button, which you always can use
No good tutorial.

Pro:
Nice concept.
Getting more challenging over time, because rules change
