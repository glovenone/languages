About This Game

Escape is a small 2D strategy game where you have to go through different rooms. Each rooms has a finish point and you need to manage to go to that point without dying. The game is quit simple, it can be finished very quickly depending on your jumping skills !

There are a total of 35 levels. All the levels are already unlocked and ready to be played.
You can put your own favorite music as there is none in this game.

Enjoy :)





Please report any bug at this adress : escapeinformation@gmail.com

At some point, this problem got in the way really bad: it wasn't that the platform level was very hard, it was that the controls didn't respond like they have to, so I kept dying because I couldn't control the square properly.

EDIT: scratch that. Some levels are impossible.
