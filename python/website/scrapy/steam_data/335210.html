Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We want to involve everybody in our developing process. We need your feedbacks,we want to deliver the best epic/fantasy Virtual Reality experience and we want to know what you want to see in Rift's Cave the next future ,what kind of new features you're dreaming ,what kind of abilities,what kind of puzzles or what kind of environment you're curious to explore. We think that Rift's Cave is a magical adventure,and we want it will be so for everyone.”

Approximately how long will this game be in Early Access?

“This depends of how feedbacks we'll receive and how fast we'll develop all the new features we want introduce. However Rift's Cave will be in Early Access for not more than 3-4 months.We're planning to introduce all the key-feateures as fast as we can.”

How is the full version planned to differ from the Early Access version?

“The full version will have some more key features. This early stage of the game can have some "empy moments". We want to fill this empy holes to make a more fluid game. A great part of the development for the full version will be to make more, and more complex puzzles to solve,add some magic abilities,add some more maps to explore,add something more about the plot,and of course solve all the bugs we'll find out. Our goal for the next weeks/months of development is to make the most Epic/fantasy adventure and immerse experience with the Oculus Rift!”

What is the current state of the Early Access version?

“Actually we have released the Alpha v2.0 . This version is the biggest release we have ever made with Rift's Cave.The game already have a well implemented plot,the 75% of the explorable maps,the key abilities and some simple puzzles to solve.

Of course, during the development of Rift's Cave we'll update the Oculus SDK so the performance can be improved, including the head tracking to be more low-latency with some other good results. With it we'll continue to release updates to fix bugs, adds some new features and a better experience for everybody!

The major thing we're happy about is the exploring side,since we've made several maps really cool to explore in VR. We have implemented the latest available SDK (0.4.3.1) and this alpha already support the Oculus Rift DK2 with all its features( Positional tracking,low-perstistence etc).”

Will the game be priced differently during and after Early Access?

“This is something we're thinking about. Obviusly an early access buyer will try the game before the others but with bugs and incomplete features. It's a chance that the Early Access will be priced slightly different from the final version.”

How are you planning on involving the Community in your development process?

“We need your help to better understand what we can improve. We already know there are a lot of bugs to fix and we're working almost 24/7, but in the meanwhile it will be really useful to know what kind of features you want to see in Rift's Cave.

The community will have a big role during this developing stage. We want involved everybody to help us to make some other key features of the game like: More complex puzzles,More abilities,More maps to explore,something more about the plot etc. The community will have the chance to help us to shape Rift's Cave and make the other key features we haven't already developed.

Do you want more complex puzzles? Do you want a deeper story? Do you want new magical skills? More maps to explore? More enemies? Less puzzles and more exploration? Just let us know and we'll try to follow what want for Rift's Cave!

We need to understand what kind of puzzles are less intuitive and we need to know if you think we need to chance/add/remove something to make it more understandable. Since Rift's Cave is also a puzzle game there are some stuff that you have to work out yourself, but sometimes it can be hard to premeditate if something will be too much hard to understand, so we need your help!

Also,if the community will be unanimous to unlike some key features of Rift's Cave we'll think about remove it to make something more enjoyable.”

Early Access and Oculus Setup Guide

IF YOU ENCOUNTER ANY FRAMERATE ISSUES, STARTUP ISSUES, OR OTHER, PLEASE VISIT HERE FOR A FULL LIST OF SOLUTIONS

HOW TO CONFIGURE THE OCULUS RIFT FOR RIFT'S CAVE

  • Check "Direct Mode" in your Oculus Config Utility and then launch the game through Steam.

PLEASE KEEP IN MIND THAT THIS IS AN ALPHA IN EARLY DEVELOPMENT


You WILL encounter bugs, and you may need to relaunch the game if needed. However we have been testing the game for a while and we've never experienced anything that required a restart, so you shouldn't either!

The game is under FULL and CONTINUOUS development. We will release updates as fast as we can. Mac OSX and Linux support will be added in the future. Please be sweet with us and help make something really cool! A lot of feedback is really appreciated.

About This Game

THIS GAME REQUIRES THAT YOU OWN AN OCULUS RIFT DK2.

Rift's Cave is an adventure/puzzle solving Virtual Reality game designed for the Oculus Rift.

This game was inspired from beautiful, diverse environments and scenery - in Rift's Cave you'll be able to experience such inspirations for the first time in stereoscopic view.

You'll live an epic adventure exploring fantasy landscapes, solving mysteries and using your magic skills to defend yourself from what lies within. Discover epic temples, deserts, snowy forests, giant caves and much more!

Rift's Cave Alpha Features


  • Begin the adventure, find out who you are and why you're stranded in a mysterious cavern.
  • Use your magic skills to protect yourself against bloodcurdling ghosts, solve puzzles, or just turn a light when you can't see!
  • Explore for hours! You'll be projected into a new fantastic fantasy world for the first time in VIRTUAL REALITY!
  • Bizarre creatures that will be lurking around any corner.
  • Use your special view to relive ancient adventures and secrets including additional backstory on the cave itself!
  • A new level of immersion and some epic fantasy maps to explore, just dive in!

One of those uncompelling VR purchases.

I'm all for tons of VR, more than I can even play. When a new title on Steam announces it has VR support, I normally investigate and see if it will work with the DK2 (Yes, beware as Valve is too incompetent to properly label VR titles; you must do your own verifications in order to make sure the game actually works with VR, even if Steam says it should... it may very well not - Steam just want to take your money)

While this title worked virtually perfectly with the DK2, content-wise it fell short. Very short. Don't get me wrong, it looks fairly good in there (fairly, not incredibly... I was more impressed with the visuals in Frontiers) but it just doesn't seem right. The scales are off, the LOD lacks and as a showstopper, some basics like camera clipping values are wrong.

I stopped my run when I recognized something I learned in Unity on day 1: I could see a big diamond-shaped hole in the skybox. So we didn't change our default Oculus camera rig values. Hum. Quit.

I regret this purchase. I find it alarming to read things like "must own for DK2 users". Your bar must be so low in the must have dept.

Originally I posted that the game made me sick due to framerate issues and that the game would quit after you hit the start button on an XB360 controller.
