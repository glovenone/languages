About This Game

Colortone is a first-person puzzle-platform video game. You are trapped in a room from which you should escape. You don't have any abilities. The only thing you have is your brain. By using it you can get out of the most difficult situations. The room becomes more and more difficult each time. To reach the exit you will need to solve various puzzles.

Pre-Completion Review

Initially I found out about this game through an Antichamber thread, where Californ1a mentioned its similarity. I am now finding other games like Parallax to be more similar to Antichamber than this one. It seems the developers didn't take advantage of the early access that they were granted, as some quality features were lacking, including but not limited to:
  • I thought the game had no sound because the volume slider was all the way down by default.
  • But there's more to it than that: there are almost no highly noticeable game sounds nor interesting music, and that is a big key to making any game memorable!
  • Controls are a little weird when you first meet up with them, as there is no context sensitivity.
  • The settings are minimal in-game because they've been moved to a dialog box outside of the game (eh, that's just the way the Unity game engine works though :P)
  • Worst of all, there appear to be situations where the player can get trapped and not be able to pass the level. These can occur as early as level 2. This is really annoying for a puzzle game. At the least some sort of reset switch would be nice.

Post-Completion Review

OK, now having completed the game, not much about my opinions above have changed. I think the reason the game lasts a couple of hours for about a hour's worth of content is the game's bugginess. In level 13 for instance, I got stuck for the longest time because I didn't realize one ball was colored because it appeared white. That made a huge portion of the puzzle redundant after that simple realization. As I said before, not enough debugging and testing was done on the game.

Also, along the way, a friend told me that the newly added achievements are broken.

Rating: 4/10. It's not as bad as some games, and the concept is interesting, but it's rather poorly executed.

Colortone

Colortone is an interesting Puzzle Adventure Indie game where you use colors to solve different puzzles. I will provide a TL;DR at the end.

Gameplay

Currently there are 14 (15 if you include level 0) levels to play through. The first few levels could be seen as tutorials to show you the different features and ways to beat puzzles within the game. Everything within Colortone is linked with using colors, whether it be to finish the level, destroy a wall, or to make a platform. There are 6 colors within Colortone which are the 3 primary colors (blue, red, and yellow) and the 3 secondary colors (purple, orange, and green). There are 2 types of object you can color according to which dab of colour you put it in, a small sphere and a board. You can dip these in individual colors or you can dip in one, then another, to create a secondary color. The game uses an interesting platforming idea where you place your boards on certain colored rods and if the board matches the colored rod then it have created a platform for you to jump on. There are also machines which use the sphere colored with primary colors to create its secondary colors (Example: blue would create green and purple), secondary colors create their individual primary colors (Example: green would create blue and yellow), and a machine which will take the color of the sphere and put it down so you can use it (Example blue means you get blue). There are certainly a lot of good puzzle elements which can be utilized within Colortone to make sure it is a very pleasant experience.

Graphics/Sound

Colortone has an amazing soundtrack that is able to easily ease you into a calm mood to play through and enjoy the game. The graphics are fairly basic, nothing that stand out that much. The game will be mostly white except for the colors seen in walls, platforms, spheres, machines, light beam, and everything you can use/have to go through to beat the level and lava.

Pros/Cons

Pros

+ Good concept
+ Great use of color within puzzles to solve them
+ Good soundtrack
+ Fun gameplay
+ Options for colorblind to allow for them to enjoy this game

Cons

- Very limited replay value (only 15 levels)
- Beatable within 2 hours
- Some levels have a slight problem where if you die with an object and lose it, you are unable to obtain it and have to restart
- Unable to run on Windows 10 (at the moment)

Suggestions

These are my personal suggestions to the game that I feel will change the game for the better.

- An in-game editor to allow players to create their own levels along with a sharing feature to play other people's maps
- More campaign maps

Final Thoughts

I was able to enjoy almost all of my time spent within Colortone, but I was disappointed to find out that there were only 15 levels with nothing else. This game would be drastically improved if it had an editor with the map to create custom maps.

TL;DR

Colortone is a very interesting and fun to play puzzle game that utilizes colors. The game is currently around 2 hours to beat with only 15 levels. The soundtrack does a nice job in immersing you into the game and makes for a more relaxing experience. If you like the look of the game and have the $5 lying around you might consider purchasing this for yourself as it's a good experience with everything that is within the game so far.

General Info
I originally picked this up when I heard about it during the Co-Optional Podcast's releases section when TB mentioned it looked Antichamber-esk. With me being one of the few people who have hundreds of hours in Antichamber (speedrunning/routing), it looked like an obvious game to pick up, and upon buying it (during it's early access period), noticed it had a "Speedrun" mode button on the main menu, which appealed to me. At the time, it wasn't a functional mode, and was just a placeholder button. I livestreamed my first play of the early access version, and promptly posted here on the community discussions what my thoughts were (along with some bugs).

Developer
The dev is very nice and responsive in the community discussions, however this is where the game starts to get a few demerits from me. I realize making a game can be time consuming, and plenty of other things can be much more important, but the dev had specifically assigned dates for each new level to come out. This schedule worked fine for the majority of the levels, however near the last few levels, the dev missed releasing about 3 or 4 levels in a row, constituting about 2 months of no updates, with no word from the dev about when updates would come. The only information was via this thread on the community discussions. The dev also does not maintain any active news/announcements posts or changelogs so when the game does update, you have no idea what has changed, and you can't look back at any "previous" changelogs because none exist.

Schedule
Mid-August was meant to be the release for the first level that was skipped, and as seen in the thread linked above, the dev said that all the final levels as well as speedrun mode would be finished by the end of September. Nearing the end of October is when the update finally arrived, without speedrun mode, and with two broken levels. The bugged levels were promptly fixed, as I have already said the dev is responsive on the community discussions.

Bugs
Sure there are bugs, but nothing too major. Any major bugs I've seen (and posted about) have been promptly fixed by the dev. Again, as I said above, the dev is quite responsive for this kind of stuff. The only main bugs I can think of currently are that some objects can get stuck inside walls/floor, but it can be looked at as a quirk of the game and just takes some getting used to how to control yourself when carrying an object. I've found that the controls themselves are fine for my tates, as opposed to what some of the other reviews say about the right/left mouse functions, just takes some getting used to.

Speedrun Mode
There is no information regarding speedrun mode at the time of posting this, and last I checked, the button in the main menu has been removed, and the game has come out of early access, so I feel as though the dev has dropped the idea completely.

Conclusion
$5 is pretty cheap, so I would recommend it, and the game itself as-is right now is still pretty great for that price, however I do feel like there are a lot of things the dev could have done better during the early access time period, such as posting changelogs (or even having version numbers to begin with), or news posts about any delays or changes of ideas/direction.
