About This Game

Puzzle Station offers two game modes: Arcade and Puzzle. Both modes have 50 levels.
In the Puzzle Mode you are given a limited amount of drops and your goal is to clear the playfield before you run out of moves.
If you are looking for a more casual game mode and just want to "zone out" the game offers an Arcade Mode where you can just blast away and hunt for the high score.

System Requirements

    • Memory: 1 GB RAM
    • Graphics: DirectX 7.0 compatible

    • Sound Card: DirectX 7.0 compatible
    • Additional Notes: No game controller support

Basic Information
Title: Puzzle Station 15th Anniversary Retro Release
Genre: Casual, Indie
Developer: Ninai Games , YeaBoing
Publisher: YeaBoing
Release Date: 14 Jul, 2000

Basic Overview
For starters I am not one for puzzle games. But this puzzle game I really enjoyed, it is surprisingly fun to play, although it lacks a few basic things to keep you interested enough to play it for hours on end. It has some okay music, and generic sounds. But for a puzzle game you only really focus on the music and game play.

Game Play
When I first played this game it reminded me much of the old game Dr. Robotnik's Mean Bean Machine, with the similar style of game play. There are two types of mode in this game. Arcade mode, and puzzle mode. This does add a little bit of replay value to the game, but still only really kept me interested for about 2 hours before it got a bit boring and started to become a little annoying. The controls you will be using are Ctrl and arrow keys. I'm really not sure why they chose ctrl to be their main button to select things in the main menu and to use in the game. I did have a few issues with the ctrl button, for one it's right next to the windows key for me, and pressing that minimises the game. If you then try to go back on the game it crashes. Really annoying, and couldn't find anyway to change the controls.There are two types of game modes on this game. Arcade mode is basically connecting 4 of the same coloured blocks, that will get rid of them. Each level you have to get rid of a certain amount of blocks to pass the level in the time limit that is given to you. The first few levels that I played I didn't even realise that there was a time limit that I was up against. Obviously I found this out after a few levels, and this time constrain gets pretty tight on the later levels that you get into the game. There are a few elements to this game, that are again kind of similar to the old Dr. Robotnik's Mean Bean Machine game. There are brick type blocks that if your 4 blocks (or more) touch the brick block it will take it out with them, or they will fall if they are not taken out and if they have space to fall. If however there are solid blocks, they cannot be gotten rid of and doesn't have gravity to it either. These two simple elements of the game make this game get harder as it goes along. Along with the time constrain it makes it a nice challenging game that has a nice difficulty curve to set you into what the game has to offer, and doesn't drastically increase the difficulty which is really good for this type of game. The other mode that there is is called Puzzle mode, it is what it says it is. Basically you have a limited amount of turns to drop the blocks, and you must clear all of the blocks to pass the level. I felt like some of the later levels where way too easy then some of the levels before that one, this makes me feel clever but kind of defeats the object of a puzzle game. Both of these modes have a nice little kind of collectible, you can unlock different colours of like the hand held console thing that you play these modes on. Nice little feature.
Score (Out Of 10): 8 - Good variety of modes, Arcade and puzzle, although in puzzle mode some of the puzzles seem too easy later on and too difficult early on. Also I would defiantly change the main button to select things from ctrl to something else

Graphics/art style
The Art style of this game is cute, and fits in with the whole puzzle game on like a old school hand held device. There isn't really much else to talk about for the Art Style of this game.
Score (Out Of 10): 10 - Very fitting graphics

Music & Sound
The music for this game is not bad, although there is only one track that repeats itself over and over again. It really needs to be varied when it comes to the music because after about a hour of listening to it I decided to just mute my sound so I wouldn't have to listen to it anymore. As for the sounds they are pretty generic for this type of game, nothing else really to comment about or say.
Score (Out Of 10): 6 - Really annoying music if you're playing in a long session, sounds are pretty generic.

Conclusion
I defiantly recommend this game to puzzle fans, and people who don't even like puzzle games. I myself I don't really like puzzle games but enjoyed my time with this game. The game play is really fun and satisfying, and if you can get over the awkward ctrl button and repetitive music this game is really fun.

Possible-medical-fore-warning:
There has been some reports + newer-reviews mentioning something about the "current-built" of this game "flickering" quite a alot on some of the machines. While this doesn't make this game unplayable (as of writing this), some people might experience nauseating or any similar effects.
Personally, I do indeed see some consistent flickering. But I don't exprience any medical-side-effect, at least not yet.


Technical-fore-words:
It is fair to warn right away that this is indeed a very "as-it-is"-release of the originaly year-2000-build of this game, which is also mentined on one if the first pre-title-images. So there are no build-in tweakings in options-menu for windowed-mode, resolution-changes, or various other "modern" tweaking-options aside wheteher you want to turn of the sounds or not most importantly.

That being said, you're going to get one of the least amount changes suffering re-releases on Steam, only chnage being that it works on (at least most of the ) moderm machines.


You're also unfotunately forced to use another 3rd-party screen-capturing / shotting-software since due the age and un-modified nature of this game, the Steam-overlay won't be working (at least by default).

--------------
--------------

As for the game itself, "Puzzle Station" is the non-product-placement "off-spring" of Ninai Games (before Detonium Interactive) of "Drop Mania" (which eventually was "remade" as "Super Drop Mania" for Gizmondo by "Fathammer"; then there's of couse the "Rampage Puzzle Attack"-variant also by "Ninai Games").

Like the title suggest, this "Drop Mania"-variant is much more puzzle-heavy for the most part. But considering the simplistic nature of the game-mechanics (link 4 or more of the same color-blocks to "poof"-them), I for one can forgive this even if sometimes I'd like to just watch pretty explosions "without thinking" (I.E. playing without swapping the blocks with Ctrl-button).


So, is the game worth buying?
Well, puzzle-game enthusiastics, especially those who are looking for some older titles, this is quite a good and solid one to their collection.
That's pretty much all I can say about this game, for now.

And who knows, maybe we'll see an actual new "Drop Mania"-game in future on PC.
