22 June

Major Update 1.1

The big 1.1 update is out and brings a lot of new features and improvements for the game. Besides many others, the main additions/changes are:

  • 10 New Singleplayer levels
  • New Special block: Bouncing Ball
  • Invention tool
  • First person camera (shortcut [F], only in Sandbox mode)
  • Slow motion function (shortcut [M])
  • Redesigned menus & in-game UI
Have a look at the changes in the new trailer:
https://www.youtube.com/watch?v=yCclmdqeKS0
The singleplayer levels included in this update are split into two categories in a new singleplayer campaign so that they can be played even if you haven't finished the old levels. Furthermore, each category has it's own progress so you can choose which type of levels you want to play first. The new category system also allows me to easily add new categories and levels.

The Invention tool allows you to save things as inventions in any game mode without having to leave the game and switch to the Inventor mode. This makes it a lot easier to build repeating structures and is especially useful in multiplayer games.

A major part of this update is the redesigned menu and in-game UI, which is now easier and faster to use while offering more information (like preview images for maps/inventions). In the game, I got rid of the menus at the top which you had to click on to open the tools or blocks and grouped them together. Another change is that the multiplayer sandbox menu is now integrated in the multiplayer menu and can be selected as game mode.

Apart from everything mentioned above there are a lot of smaller changes like object highlighting, performance improvements or more precise dragging of blocks. I'm currently working on the Steam workshop implementation which is about halfway done and will be part of the next update. Have fun!

15 April

Update v1.0.5

Hi guys!

It took me a bit longer than expected to finish the patch as it contains many changes and fixes that were more complicated and initially caused some problems. Here's the full changelog:

Improvements:

  • Changes in the tutorial to explain things in more detail.
  • Trigger now display connections which are over the maximum length as red lines.
  • Reduced network overhead in multiplayer sandboxes that have many special blocks.
  • Blocks that are very far away from the map will now be deleted (In order to prevent needless physics calculations and network traffic)
  • Added option to change the keys for camera movement.

Changes:
  • Increased maximum length of connections of triggers and gates from 16 to 20.

Bugfixes:
  • Fixed broken inventions in the multiplayer sandbox (If a client with a slower internet connection placed a big invention with many connected blocks, it could happen that it wasn't created the right way)
  • Fixed a bug where the camera could get stuck under the terrain.
  • Fixed a bug where the camera could become stuck in the wall separating the players in the multiplayer (with phases activated).
  • Fixed wrong tooltip on height display in the multiplayer mode
  • Fixed a bug that camera objects don't get deleted if player leaves a multiplayer sandbox.
  • Fixed "Not enough resources" text in the sandbox mode (Couldn't reproduce the bug but it should be fixed now, if it still occurs, please write a message)
  • Fixed bug where inventions with vertical rockets couldn't be placed on the floor.

I'm currently busy working on the bigger 1.1 update which will introduce several new things and will improve some of the existing things in the game - more info soon. That's all for now, stay tuned!

About This Game

Brick Inventions is a physics based building game. It combines simple blocks with realistic physics so that everything in the game can be completely destroyed. Build awesome structures, invent machines, master difficult challenges or play exciting multiplayer battles against your friends â Brick Inventions offers a variety of different game modes for all of that.

Singleplayer

Play through 25 singleplayer levels, of which each one offers a different kind of physics based challenge. Besides the normal mode, many levels can also be played in a more difficult challenge mode in order to earn medals and to unlock achievements.

Multiplayer

Fight against your friends or other players in the strategy oriented online multiplayer mode. Depending on the selected game type, each player has to gather resources, build warehouses to increase storage capacity and unlock blocks in the research center in order to build more powerful defenses and weapons. The multiplayer mode offers two different game types on two distinct maps.

Sandbox

In the sandbox mode, you have unlimited resources and can build whatever you want. As it is way more fun to create things together with other people, the game also contains an online multiplayer sandbox mode that allows you to play together with your friends.

Inventor mode

Since everything is created out of blocks, it might take long to build things that are more complex. The Inventor-mode enables you to create your own inventions, which can be saved and placed with just one click in any other mode. Invent a new machine, more efficient weapons or powerful defenses - the better your ideas, the better are your chances of winning a multiplayer game or mastering a challenge in a singleplayer game.
