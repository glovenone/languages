About This Game

The recent events in a state prison, involving Jake and the others, was a perfect moment for some prisoners to escape - you are one of them.

Prison Run and Gun is a standalone expansion to Hot Guns game but this time with more puzzles and platforming action !
Use your skills and the guns to break through 30 sections standing between you and the freedom.

If you have purchased Hot Guns you will get this game automatically for free.

http://store.steampowered.com/app/376910/

Review written on Sunday, March 20th, 2016

Review
This Game is not just an expansion to Hot Guns, it’s also a Masterpiece itself. The graphics are kept in a high resolution retro version which is totally fine and makes this Game a must have for every retro fan. The Game itself is a platformer with a well-made full controller support (tested with Xbox 360 Controller). The character (a prisoner) moves smooth and I did not notice any bugs while playing. The difficulty of the level slowly increases in a fair and enjoyable way over the 30 levels you can play.
The goal of the Game itself is to escape from a prison which is split in 30 sections. You will find many different items which can help you to escape from the prison. For example Guns and Helmets. The Game principle itself is very simple but makes the Game super fun. Also if you think the Game is too easy for you, you may also try to get the hidden Keys in every level.
I totally recommend to buy this Game now (Since it has a -20%) because in my eyes the Game is worth far more than only $2.

Rating:

8.8/10

Pros:
  1. Well-made Platformer
  2. Enjoyable puzzles
  3. Full Controller Support
  4. A lot of different items
  5. Works also on older devices
  6. Well-chosen music retro music

Cons:
  1. No achievements
