About This Game

After blasting all the beasts a new threat has arisen. The same technology that was used to destroy the beasts has evolved and created cyborgs! There is no other way but to detonate all of them, and you are just the right guy for the job. So, arm yourself with something that goes BOOM, get the right attitude and let the cyborg detonation begin! Don't let the bastards wipe us out, we should be playing computers, computers shouldn't be playing us...

5/10

I did not play for very long before I realized that there are a few issues with this game that made it unenjoyable for me personally.

1. Some of the sprite boundaries are larger than the sprite. Meaning that some edges of objects/characters don't ever touch because the invisible bounding box around the 2D image is larger than the image itself. This can make the object collisions look poor.

2. The music is well intended, but ot quite fitting for as casual as this game is. The music is a little too upbeat in my opinion.

3. At first I did not realise that a bomb explosion could travel through objects like metal and stone. And although it doesn't make scientifical sense, it seems to be necessary to clear a level sometimes in this game. I found out on accident.

4. The user interface is a little too flat; with oversized buttons that dont feel right to me. And the game isn't designed for full screen so it looks a little grainy.

5. There are enough bombs to clear a level without any challenge (or at least in the first few levels). The only real challenge is to get a perfect score. Example: there are 2 bombs and 2 enemies. Sure there must be a way to get them both in 1 explosion, but its no challenge at all to use 1 bomb on each of them and pass the level.

6. I blasted through the first set of levels on the map in a few minutes. Way to fast.

7. Some levels start out saying something like "don't let him fall" while there is a character sitting on a platform. Basically you need to place bombs to explode the enemies without tipping over the platform with the good guy on it. Trouble is, before I can do anything, the structure falls over and 'he' falls with it.

Overall, I was hoping to get something as cool as World of Goo or Angry Birds, and this isn't even close. I feel like this is a bad port that was meant for an iPad.
Only reason I give it a 5 is because it is playable and it does deliver the minimum expectations.

But if you love to support indie games and you are all out of casual games like this to play, then give it a shot.
