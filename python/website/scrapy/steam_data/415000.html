I have completed Chapter 4 (~70 videos) and i'm having a great time learning through this tutorial.
I don't use the launcer to launch the videos, so the 4.4 hours is of steam cards idle time.
The videos and the Source scripts are well sorted and is easily accessible.

You can directly stream the video's through youtube for free. Yet i bought this because, It is much more convenient for me to have offline videos and run them through vlc. and you know support the tutor.

I Started Game development through GTGD S1 and stopped it after 50% because Unity 3 was outdated.
then i jumped to S3 with unity 5, after it was released.

So GTGD S3 unlike it's previous seasons has Tutorial Videos of small duration ranging from 3 to 25 minutes.
The tutor manages to make sure everything you learn is in a simple and efficient manner.

I have now completed
1.Basic Course.
2.A Game Management System
3.Player Health Managerment
4.Inventory Management system.

I come from a programming background, I know C, C++, and now C#. (I'm a student myself)
It will be great if you know the application of logic with programming, otherwise you will learn it anyway.

Also check this link where you can find some of the submissions of fellow developers which might motivate you to start making your own game

http://forum.unity3d.com/threads/unity-5-fps-tutorials-gtgd-s3-advanced-first-person-shooter.360105/

All in all 10/10
