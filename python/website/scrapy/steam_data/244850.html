About This Game

Space Engineers is a sandbox game about engineering, construction, exploration and survival in space and on planets. Players build space ships, space stations, planetary outposts of various sizes and uses (civil and military), pilot ships and travel through space to explore planets and gather resources to survive. Featuring both creative and survival modes, there is no limit to what can be built, utilized and explored.

Space Engineers features a realistic, volumetric-based physics engine: everything in the game can be assembled, disassembled, damaged and destroyed. The game can be played either in single or multiplayer modes.

Volumetric objects are structures composed from block-like modules interlocked in a grid. Volumetric objects behave like real physical objects with mass, inertia and velocity. Individual modules have real volume and storage capacity.

Space Engineers is inspired by reality and by how things work. Think about modern-day NASA technology extrapolated 60 years into the future. Space Engineers strives to follow the laws of physics and doesn't use technologies that wouldn't be feasible in the near future.

Space Engineers concentrates on construction and exploration aspects, but can be played as a survival shooter as well. We expect players will avoid engaging in direct man-to-man combat and instead use their creativity and engineering skills to build war machines and fortifications to survive in space and on planets. Space Engineers shouldnât be about troops; it should be about the machinery you build.

Space Engineers is one of thoughs games where you don't see how it could be all that fun until you try it. Where to start this review is almost a task by itself. The game visually is superb even more so with the lastest update to DX10 the only real downside is the amount of computer power required to play this game. However this is a small price to pay when you look at the amount of game time you can get out of this small purchase. I bought this game for myself and a best friend and we hit the ground running no tutorials or anything. I can say after putting over 600+ hours into this game the money was well worth it.

With the most current updates the game has progressed in a matter which I would have never seen it going with the addition of planets I was totally blown away I never pictured this game growing into what it has. This being said I have bought another four copies for more of my friends just to enjoy the game with some of my friends. With the mods and scripts you can run on this game it can really open a new world of options for the game.

The gameplay itself at first seems to really have a slow progression even more so if you are in a survival game however you will still put five or six hours into a game before you even notice how long it has been at which point the game has started to progress faster. After a few days once you have managed to get a base or ship in a state where you can mass produce stuff your progress will just take off to a point where you have a hard time keeping up.

One thing that makes this a special game is when you play it single player or even coop with only a few people its a fun relaxing game however if you get on a coop server with twenty or more players this game opens up as a whole new beast you will find yourself in a survival against other players and pirates in a race for resources. I have never played many games where the game itself changes based on how many people are playing the game with you. However this game truely does evolve and take on its own life as the server gets larger the only real downside is the amount of power your computer must have to run this and the lag.

This being side the game is being updated weekly without fail even on holidays it still gets an update the team is amazing and has really progressed this game a long way. As we progress closer to the netcode updates I expect the lag will really start to settle down and the large servers will start to popup more. Without a doubt this game is worth picking up even if its not on sale you can be rest assured you will get your moneys worth from this one.

It's like Lego in space!

You can create spaceships and land vehicles and explore, fight and colonize.

But it's so much more if you want it to be - there is also power management, weight distribution and proceduraly generated asteroids and whole planets and moons! Yes, Space Engineers did it first, and although they do rush some of the features in order to be first, I really believe they are pushing the game industry forward with their ambition.

If you want a big challenge - there is a punishing Survival mode - where you have to mine, produce and assemble all your creations. Imagine having to actually create all the blocks before assembling them. This makes you design efficient, realistic vehicles and spaceships.

Some of the players complain that the game is what it openly states it is - an Early Access game. Some features simply don't work yet. My solution to this is to play with the features that work perfectly and I couldn't be happier. The team gives weekly updates to the game, and there is a very strong modding and blueprints community that pushes the format of this game to the limit.

It's been a fun ride so far and I believe there is a lot in store for this game in the future. It's ambitious, it's serene and I keep returning to it - playing it for years now. Definetly worth your money!
