Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“Bombernauts started off as a rough prototype posted to the web in 2013. The community was very important in the early development of the game, and adding people's suggestions was really fun. We'd like to end the development of the game the same way, and get the community involved once again.

Also it's our first multiplayer game, and we want to make sure it's as bug-free and balanced as possible before release.”

Approximately how long will this game be in Early Access?

“"About 6 Months". But it could go longer, and probably will. So more accurately would just be, "Until it's done". We're going to be taking suggestions and seeing demand for stuff while it's in early access, so if there's a lot of demand for a feature we didn't expect, it could take longer. It's also really notoriously difficult to actually estimate how long game development stuff will take. We originally thought the game would be out in 2014. Ha.”

How is the full version planned to differ from the Early Access version?

“The UI needs an overhaul, and with that will come easier options for game customization. The level editor could use some touching up as well. There's also a walrus referee character we haven't added yet, and some story/setting related stuff to flesh out the game world a bit more. Plus some more game modes and powerups and some stuff for more flexible custom game support. We also want some more cosmetic settings for robots like hats and alternate wheels and stuff. Plus, whatever else people are wanting during its stay in early access.”

What is the current state of the Early Access version?

“Bombernauts is fully playable right now, and around 80% the way to a "version 1" official release.”

Will the game be priced differently during and after Early Access?

“While in early access, purchasing Bombernauts gets you an extra copy to share with a friend.”

How are you planning on involving the Community in your development process?

“We're going to be taking suggestions from the community for new powerups, game modes, cosmetic items, customization options, achievements, game balance, and more. See the steam community forums for more specific information on how you can help!”

HOW THE SING COMMANDS WORK
First of all, in sandbox mode there are no limitations or cooldowns songs, so this is the mode you should be in to test these! In a real game, these commands have a 120s cooldown, and can't be longer than 10 seconds or have more than 100 notes.
Try these 2 sample commands first:
-sing {4,2,0,2,4,4,4,x,2,2,2,x,4,7,7,x,4,2,0,2,4,4,4,4,2,2,4,2,0}
-fancysing {(1,0,500),(0,2,250),(1,-2,400),(2,0,1000),(1,3,500),(0,2,250),(1,-2,400),(2,0,1000)}
-fancysing {(0, 3, 0)(0, 8, 150)(0, 3, 0)(0, 8, 150)(0, 3, 0)(0, 8, 150)(0, 3, 0)(0, 8, 150)(1, 1, 0)(1, 6, 300)(1, 1, 0)(1, 6, 300)(1, -2, 0)(1, 3, 300)(1, -2, 0)(1, 3, 300)(3, 1, 0)(3, 6, 300)}

"-sing" is a list of pitches (with 0 being C3). Valid ranges for pitches are -40 to 40, with an "x" indicating a rest. Each note lasts for a quarter second. Simple songs are very easy to make with this command, however if you want better control over timings you'll want to use the fancysing command

"-fancysing" is a list of sets of 3 numbers, each representing 1 note. The format is (sound, pitch, delay). Sound is 0-4, with each number representng a different sound clip to use. Currently it's 5 different "la" sounds, however we may add more in the future. Pitch is the same as in "-sing", however "x" as a rest is not enabled here (just use a delay on the previous note, or a pitch outside the valid range). "Delay" is how many milliseconds to wait before playing the next pitch. You can use 0ms delay to play notes at the same time for chords.

In both commands, the commas and parenthesis are optional, however if you use spaces to separate things the note list should surrounded in brackets {} or quotes ""

I expect people to create and share all sorts of crazy sings they come up with. Eventually I will integrate this into the game better so people can save them and play them automatically as a taunt without having to paste stuff into the chat, but for now this is pretty fun as is.

Early Access Review
Posted: 18 September, 2015
Bombernauts
Developer: Eyebrow Interactive
Publisher: Eyebrow Interactive

Note: A key for this game was provided by the developer for the purpose of writing an unbiased informative review. No other compensation was given.

Introduction
==========
Bombernauts is a online multiplayer arena game with fully destructible voxel terrain. Think Bomberman, only a lot more fun; Your character is a little buggy-man thing, armed with tons of bombs and a baseball bat. Bombernauts is an incredibly fun game with friends, but therein lies it's largest problem: the playerbase is currently non-existant. While this isn't the game's fault, as the game itself is solid with a great concept and little to no bugs, it currently does not have any singleplayer or bot options for players to mess around with while they wait for other players to battle. However the devs are currently working on adding these features, so we should see the playerbase rise considerably after these are implimented. Now let's take a look at the game itself.

Gameplay
==========
Remember Bomberman? Well, this game is Bomberman without the grid layout on it's maps. You control a small 3-wheeled buggy-man-thing (let's just call it a buggy), that's armed with an infinite supply of bombs and a baseball bat. You can play against up to 16 other players (4 is the recommended number), where the objective is to blow them all up! Sounds simple, but it is insanely fun. You throw bombs in an arc, and you can aim how far your throw will be. Bombs explode a couple of seconds after landing, and if you hold off before tossing them while aiming, it'll cause a larger explosion...sort of like powering them up. The baseball bat can be used in a pinch to knock enemy bombs away from you, even before they land! Batter up!

At times, a small drone carrying a pinata will appear. If you bust up this pinata with your bat, it will drop a random powerup. These powerups are very numerous and do a variety of things, but usually only last for 1 bomb throw. For example there is one that lets you throw a cluster of bombs at once. When this explodes, it shoots many smaller bombs out, which then explode as well. Another powerup gives you a temporary jetpack, which improves your mobility greatly. There are a few referential powerups too, that give nods to various things. One great example of this is a powerup that gives you Dragon Ball Z Super Saiyan hair, and allows you to throw what I believe to be a Spirit Bomb!

There are many, many different maps that are rotated between each match. Every map is fully destructable, even if some parts take more powerful bombs to destroy them. There is also a map editor which you can use to create your own maps. I haven't tried it out yet because I have trouble designing decent maps or creations using toolkits for games, and sometimes the controls are a bit weird. But it's a great addition.

Right now the game is multiplayer/co-op only, but as I mentioned before finding people to play with is a bit of an issue. If you want to play against others, it's recommended to get a 4 pack and give the extras to your friends who want to play, or host a game and hope others join, or advertise it on the game's forum.

Graphics
==========
Bombernauts is fairly simplistic, but the visuals are very crisp and look nice. The maps consist of mostly cubes and structures built from them, which can be blown up as much as you please. Some parts of each map react to explosions differently; for example some will catch fire after an explosion and turn to ashy blocks which crumble when touched.

Your player character is a little buggy thing on 3 wheels with goofy eyes. These look pretty comical and adds to the charm of the game. Each buggy can be fully customized with many color, eye, and wheel options.

Explosions look great. While not realistic with flames going everywhere, they are instead a shower of vibrant stars and smoke which fit the game's comical theme very nicely.

Sound and Music
==========
The bomb sounds effects are extremely satisfying. They work very well, and mesh well with the visual effects. The music is simplistic like the graphics, but it gives the game a light competitive feel, which helps as this is a competitive game at its core. It's pretty much the same tune repeated all the time, but you never really get tired of it.

I feel like the game could use a little more work in this area. Possibly a few new tunes and some more unique sound effects. What they have now is good, but it can only get better with a little time and effort!

Performance
==========
I had zero issues with Bombernauts. It ran very smoothly with a great framerate of 45-60FPS. I recommend a controller, as the keyboard controls can be more of a hindrance than help. The game loads up pretty fast, enabling you to get into the action quickly. I didn't see any random crashes, so that's a plus.

Final Thoughts
==========
With a little more work and exposure, Bombernauts could be a great multiplayer title. Bombs and powerups everywhere only add to the fun and mayhem of each game. With plenty of maps to play on, as well as a map creator, there is literally no end to the fun this game could bring. Recommended if you are looking for a game to play with friends!

Bombernauts is essentially what you'd get if you bred Super Smash Bros. and Bomberman. Easy to learn, difficult to master, this is probably one of the most fun multiplayer party games on Steam. The drop-in/drop-out lobby system makes it extremely easy to get a match going and the zillions of items, power-ups and maps makes every round feel fresh and exciting.

At the moment, the only negative is that the game has a very small playerbase which makes it difficult to find lobbies outside of peak hours. That said, it's still very playable with only two or three players, so if you can find just one friend to play with you will still be able to have fun. Hopefully this will stop being an issue as more people find out about and start playing this great game.
