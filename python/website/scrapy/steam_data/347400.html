We're back! + Update 8

I'm pleased to announce that ownership of 001 Game Creator has been transferred back to SoftWeir Inc.

For those out of the loop, we've gone back to our original name "001 Game Creator". In addition, I'm also pleased to announce the Steam build has been updated to the latest version: v1.017.001

As always, you can view the full change log here: http://steamcommunity.com/app/347400/discussions/0/481115363858417321/

Aside from branding/logo changes back to "001 Game Creator", we've also fixed a number of reported bugs and added some exciting new features including: Sticky/Platform Physics, New Platformer AI and Ramp Tile-Sets for the Platformer Template...Basically, a lot of neat platformer stuff to really flush out that template. We hope these changes will encourage more users to create platform games!

We've also added support for multiple gamepads and completely overhauled the Input Trigger's Control Combination window to be far simpler and easier to use, with different input types being split into their own categories, instead of just one big list.

Lastly, THANK YOU ALL for your support during this difficult transition! Your patience and understanding have been admirable and we couldn't have gotten through this without you and your support! :)

About This Software

Create games without limits
001 Game Creator prides itself on offering users a diverse set of tools and features to create any type of game you can think of:

  • Intuitive map editor featuring dynamic lighting, atmospheric weather effects and parallax backgrounds/foregrounds – with no restrictions on map layers.
  • Fully customisable interfaces including (but not limited to): heads-up displays (HUD), loading screens, menus and dialogue boxes.
  • Built-in graphics editor, allowing users to create their own sprites, tile-sets and items without needing to use external third-party solutions (of course you can still use external tools and import graphics if you prefer!)
  • Flexible camera controls to set the stage and provide both 2D and 3D perspectives, with full camera scripting events and 3D model/animation importing.
  • Sophisticated sprite customisation, with visible clothing and equipment changes in-game.
  • Built-in artificial intelligence and actor routes, allowing users to create complex movement patterns and fine tune character behaviour.
  • Support for up to 7.1 audio surround sound with built-in sound filters to control volume, pitch and speed variance.
  • Powerful custom gestures for creating unique multi-finger triggers for mobile devices.
  • Informative script and game debugging tools to step through events and monitor changes as they occur in real-time.

Graphically Assisted Scripting
001 Game Creator offers creators the ability to craft intuitive and imaginative games without any prior coding knowledge. This is especially beneficial to artists and designers who prefer a more hands-on and visualised approach or developers who don’t wish to spend hours bashing their head against a wall. Unlike traditional Event-Driven Programming applications, 001 Game Creator offers a huge set of built-in events, with the ability to also create custom events for specific tailor-made functionality just for your games.

PC, HTML5 & mobile development all-in-one
001 Game Creator allows developers to build their games for Windows PCs (.exe), HTML5 (.js/.html) and mobile devices (.apk) all-in-one! Unlike other similar engines, 001 Game Creator provides developers with the freedom to create their games for any and all platforms available without paying anything extra. Best of all, there are no additional license or revenue fees for selling your games - so if you’ve purchased a copy of 001 Game Creator you can sell your games without having to share the profits with us!

The possibilities are endless
001 Game Creator unlocks the doors to a wide range of possibilities. Make any game you can think of, including RPGs, shoot-‘em-ups, racing games, fighting games, simulation games, platformers, tower defense, card games, and even 3D games! If you can imagine it, you can create it with 001 Game Creator.

This is a great game engine. Sure, the UI is a bit outdated, not very intuitive, and overall it has an underground/DIY vibe that doesn't match the shiny rebranding - for now, because development is very active.

Don't expect the software will make all the work for you. It's not a game generator, it's a game engine. You will need to learn how to use it. And yes, it will require programming. Don't believe the "no programming required" crap every publisher throws at you. Even if it's visual scripting, it's still programming. You will need to think. A lot. And you will need to design your game. Nobody will do that for you.

But with a little patience, you can make wonders. Coming from Clickteam Fusion (which is also pretty good, but very different), I bought GG Maker for its 3D features, and I'm really happy with it. I wanted to make a 3D adventure/exploration game, and it has everything I need. Basically, with a bit of tweaking, I was able to create my own 3D tilesets, and use it as a voxel game editor.

It's a rather RPG-oriented engine, but with a bit of creativity I'm pretty sure you can use it for anything. GG Maker seems very versatile. I find the visual scripting system quite refreshing after Fusion's neverending columns of checkboxes. And the community is awesome. Here on Steam, and on http://engine001.com you'll find a team of very dedicated and helpful people.

About the price tag: please, shut up. You can export your games to Windows, Android, and even to HTML5. You can run 3D games in your browser. Do you know how much similar exporters cost for other engines, like Clickteam Fusion or Game Maker? Pretty much the price of GG Maker, except you also have to pay for the software, of course. Do your maths.

Yes, I definitely recommend GG Maker.

As a beginner in game development, I recommend this software to anyone who wants to learn to make a variety of games, with a rather easy to learn interface and scripting system. While some say the interface is 'outdated,' It is still easy to look at, and most importantly use! This program comes with extensive documentation, so you can always have something to look to if you forget what does what.

The mapping system is very nice. There is a small learning curve when it comes to the layers, but it is easy to get past. The lighting system is wonderful, and easy to use. you can set the color, radious and even effects like a strobe effect to it.

Can you make your dream game with this software? Of course! Currently there isn't any online multiplayer that can be accessed easily, but you can make old-fashioned local multiplayer games with split screens and multiple controls. (up to 4 players, not bad!) You can make tower defence, rpg, action, platformer, card, board game, racing game, and a whole lot more. If you are willing to put the time into learning this engine, (which shouldn't take long) you can most definitely make your dream games.

What did I dislike about GGMaker? The lack of built-in tutorials. While it brings you through the basics, I'd love to see more on advanced topics, and taking some of the communities tutorials implemented.

If you want to try-before-you-buy, go to the GGMaker website and click on the 'download trial' button. And you can go to http://www.engine001.com/tutorials.htm and look through to see what you can learn. They also have a forum where you can ask questions if you are having difficulties.

My Rating: 8.5/10
