The envoy will be trapped and there is no way out for him.

It`s quite difficult to complete this level, because you need hack the terminal which is charge of protecting lazer fields. The first person who can hack the terminal, will get money reward. Stay tuned!

This is the game currently: You walk through a few halls, go to a terminal, type login & password information (see the discussion forums), then alter some text. If that's what you look for in a game, then by all means buy it!

The publisher/developer states "at the moment this is an intervention pilot version. Anti-game or a game-contest with a great challenge". In other (English) words, this is not the actual game that 'Envoy 2' is supposed to be.. it's not even remotely like 'Envoy' which itself is a 2D shmup. Also, the developer on that game was "NCE Software Games". Here it's the same as the publisher.. so will it actually be anything similar to Envoy when this "intervention pilot version" is replaced? (IF it's replaced) Who knows! The fact is, right now it's not an actual game. Oh, and it's not an Early Access title either..

Thanks Valve! As always we can count on you for protecting us from scams like this!

*Note: in-game time was less than 15 mins; idle time was 2+ hrs idling for trading cards
