Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We've been developing Clumsy Runners for 7 months already just in two, and wanted the game to be as fun and feature complete as it possible on the actual release date. Currently the game is fully playable, has a good level of replayability and decent amount of content. At the same time we see few aspects where we want to improve the game during the early access. We would love to see community feedback to get better understanding what to do in the first priority.”

Approximately how long will this game be in Early Access?

“1 - 2 months. Depends on how many things we will decide to implement or decline based on community feedback, besides the features we plan to add for sure.”

How is the full version planned to differ from the Early Access version?

“Ideally we want to implement the next things:
- Co-op mode. We made co-op mode prototype during the greenlight, but decided not to go with full co-op mode in the current release. Would love to build this mode with your help.
- Monsters currently just go through the pits which is not acceptable. Need to come up with solution for it.
- More sound effects: when you pass the level, current on the third and fourth stages, background for upgrades menu and so on.
- Animated screen in between levels.
- Better level generation algorithm. One of the flaws: level block can be generated two times in a row. We believe it is possible to make a bit more balanced difficulties tweaking level generator.
- steam cloud saves. We used proprietary Unity's saving system, and it's kinda bad. Need to re-implement staff to make cross-platform cloud saves for Steam possible.
- Add more visual effects where it's necessary to make game experience more appealing.
- Linux version.
- Achievements.
- Full controller support.”

What is the current state of the Early Access version?

“The game is fully playable. Approximately 80% what we wanted to see in the game is done.”

Will the game be priced differently during and after Early Access?

“Clumsy Runners will cost 30% more if we add co-op mode.
Without this feature price will stay the same.”

How are you planning on involving the Community in your development process?

“Co-op
We would love to receive advices on this game mode. This feature will demand play tests from different people to make it cool.

Bugs
We want to have as much information about bugs as we can gather. Hopefully, there is not many of them (which we are not aware about).

Balance
Balancing even a small game is challenging. We are not an exception and need your help to point us out if we have any major balance caveats.

Graphics
We will add more graphic assets to the game. Provide us with advices if it's proper thing to have in game. How do you feel about new staff? Does it complements game in a cohesive way?

Depending on your feedback we will adjust direction Clumsy Runners is heading.
Thanks for your feedback.”

About This Game

In Clumsy Runners your ultimate goal is to run through 10 levels having only 1 life. It may sound simple, but running in the game is not easy. Alternate two buttons to accelerate your character. Make one mistake and you fall over, then eaten by the local alien creature.

Get in your ship at the end of each stage, buy upgrades to make your character stronger and continue to the next run. Reach enough scores to unlock more characters with special abilities.

Have fun, and keep your fingers healthy.

KEY FEATURES

  • Procedurally generated levels. Each adventure is different.
  • 6 unique characters to play.

  • 1 life to pass through all levels.
  • Buy upgrades and the end of each chapter. Try to build your own upgrade strategy which works best.
