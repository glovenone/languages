Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We're conscious that remaking a game such as The Ship is a delicate process, so we wanted to build the core game and get it in front of fans of The Ship to see what they think. We've made a number of minor changes from the original and would like to take player feedback on these.

Check out our Development Roadmap for details on current state of the game.”

Approximately how long will this game be in Early Access?

“Looking at the content and features that still have to added, we would expect the game to be feature complete soon. We will stay in Early Access as we continue performance optimisation, bug fixing, and looking at player feedback.”

How is the full version planned to differ from the Early Access version?

“The game is entirely playable, although not all maps and features are currently available. For example, traps and secret rooms are yet to be added.

All core game mechanics are in the game and functional (the kill loop, needs, security and justice systems), which means you’ll be able to play a match against other players and be able to find different weapons, outfits, and satisfy your needs. And murder. Murder works. Works real good.”

What is the current state of the Early Access version?

“Dedicated servers (16 players), LAN (16 Players) and Listen Servers (8 players) are available to set up.

All core game mechanics (needs, justice, security, combat, kill loop) are functional and in-game, and ongoing optimisation work is being carried out.

Bots and Arcade Mode (single player v bots) have recently been added to the game.

All melee weapons and ranged weapons are in the game. Some special weapons are in the game (explosives), but not others (syringes). Traps and secret rooms have yet to be added.

5 maps are available, with more due for release shortly. The remaining maps will come later in development.

Hunt, Elimination, and Deathmatch game modes are available and working well, although there are some performance issues due to lag.

Some features aren't currently available (traps, secret rooms) and these will be added throughout the course of Early Access.”

Will the game be priced differently during and after Early Access?

“No, the price will be $20 throughout. However, owners of the original game will receive a 25% discount for now.”

How are you planning on involving the Community in your development process?

“
  • All of our team are active on the Steam Community and we have a process in place to review player feedback.
  • There are several sub forums setup for players to leave feedback on specific areas of the game and report bugs.
  • We will be scheduling testing events where we ask players to look for and feedback on specific aspects of gameplay.
  • A testing branch will be maintained and players can opt-in to test new content and features if they want to.
”

PATCH NOTES
WELCOME TO 'THE SHIP: REMASTED' EARLY ACCESSThe Ship: Remasted is in Early Access and, as such, is unfinished.
No, really. Do expect whopping bugs, minor code explosions, wonky visuals, and a whole host of other strange and ungainly things that probably shouldn't be happening while trying to clobber someone with a polo mallet (e.g. sudden glitches into the 1930's -- we don't know why). You'll also find a fair chunk of important functionality is missing and some of the usability leaves a lot to be desired.

Dedicated Servers [Advanced Users]
The current build allows you to create 'Dedicated Servers' or connect to those created by others. You can install a server onto a computer and then leave the game running 24hrs-a-day for others to connect to.

If you want to have a go at installing a Dedicated Server, do the following:

A) To Download:
1. Open Steam
2. Go to Library > Tools
3. This should give you a big list of different servers and tools available. Find The Ship:
Remasted Dedicated Server (should be just under The Ship Dedicated Server)4. Right-click > Properties
5. Go to Beta tab and enter code: 'remastedtesting' to access the testing branch
6. Select the testing branch ('external') from drop down
7. Right-click The Ship Remasted Dedicated Server > install game
8. Server should install

B) To Setup Server:
1. Right-click The Ship: Remasted Dedicated Server > Properties > Local Files > Browse Local Files
2. This should bring up the folder the dedicated server is located in.
3. Open the file 'TSRDS_1' file
4. Open server.cfg with notepad++ (or a similar program)
5. Here you can edit game settings (server name, map, game type, etc.)
6. The mapcycle text file in the same folder can be edited to change what maps the server cycles through. This should be opened also with notepad++


C) Starting the Dedicated Server:
1. Go to Steam > Library > Tools
2. Double-click The Ship: Remasted Dedicated Server
3. This should open a console that shows who's joining, map loaded, etc.
4. Players should be able to join when map is loaded. (Can check its visible by going View > Servers > Ship Remasted)
5. If the map isn't visible, you must configure your firewall/port forwarding by opening ports: UDP/TCP 443, 7776-7778 on your router.

If you simply want to play a game on a Dedicated Server, all you have to do is select 'Join a Cruise', then choose a game from the 'Dedicated' or 'All' tab.

Note:
+ Dedicated Servers can handle up to sixteen players.
+ You can only connect to servers with the same version number. If no servers are visible, ensure you have updated to the latest build.

About This Game

The Ship: Remasted is a remake of the classic 2006 game, The Ship: Murder Party.

You find yourself aboard a series of luxury 1920s cruise ships – a guest of the mysterious Mr X who has engineered your presence on board to take part in a murderous game. You and your fellow passengers will be assigned a target to kill and must do so before your own hunter tracks you down. Armed with a wide array of weapons and traps, use all of your cunning and skill to dispatch your unsuspecting target and avoid sleeping with the fishes.

All aboard and enjoy the cruise.

Dedicated servers (16 players) are live! LAN (16 Players) and Listen Servers (8 players) are also available to set up. Please read our FAQs in the Steam discussion forums before you buy.

You can find our Development Roadmap here and it is updated frequently. We'll also be posting regular updates here on Steam and on our blog. Players can expect to see game updates once or twice a month at least, with the option to opt-in to test builds to check out new content and features before they go live.

Review as of 7/1/16

When I saw this game on steam, I was hoping for something that could replicate the experience of the original ship, but with a more polished engine and with better graphics. While the gameplay itself has stayed true, in it's current state I cannot recommend this game due to how unpolished it feels.

Firstly, it runs at ~50 fps on my 980ti on high settings. (Could be CPU bound, but I have an FX-8320 clocked at 4.2ghz which should be sufficient anyways). The settings menu is a total disappointment. You have resolution, fullscreen/windowed (doesn't even support borderless windowed natively), and low/medium/high. At least it has an option to select which monitor to run it on, but you can't even tweak individual settings.

It runs on Unity which is only a disappointment because I became accustomed to the actual functionality of the original Source/GoldSrc engine game. The lack-luster console and settings menus that result from not using the Source engine are a big let down in my eyes. At least it supports linux as a result.

Also, the mouse sensitivity resets sometimes during gameplay and you have to open the menu back up to fix it which is ridiculously annoying.

Right now the game just feels like a down-grade with prettier graphics. As of 7/1/16 I have to recommend the original game over this, despite having to use an external server browser to play the original it still feels like a more polished experience. The gameplay and art is there, hopefully once better settings are implemented, quirks are worked out, and at least a half-♥♥♥♥♥ attempt at optimization is made then I'll be able to recommend it, but for now the original is a much stronger experience.
