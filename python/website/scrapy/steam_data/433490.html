The Clarity Update for Super Flippin' Phones, live now

Today we're presenting an update to Super Flippin' Phones that's a bit bigger than the last patch, so we gave it a nice name! This update has two goals: provide existing fans and players with more clarity about unlocking mechanics and goal names, and make it easier for new players to getÂ acquaintedÂ with the game.

Unwrapping Costumes
We noticed our costume gifts were too subtle and often went unnoticed -- gifts are now 110% more fun *and* way harder to miss. We'll let this speak for itself:

Goals
We got a lot of questions about the goals to unlock new floors and Tiers. Since the achievement stats reflected that a lot of the content was left untouched, we decided to make some goal descriptions a tad lessÂ cryptic. We hope you like them!

Unlocking Tiers
In SFP, you can unlock difficulty tiers, costumes and new floors. All these behave slightly different when unlocked: new floors are always added to the floor pool, but new Tiers needed to be switched to manually. We've changed this; you now automatically switch to a Tier once you unlock it (the Replay button changes to 'Next Tier')

The tutorial
A dreaded part of most games, we felt that some of our messages weren't really necessary. We filtered them down to things we feel need explaining, and made others into button prompts.

Fonts
Some of the in-game fonts were rendering badly (W's, N's and M's became H's) so we replaced the font for Daniel Linssen's new m5x7 font, which is an improvement on all f(r)onts.

The volume
We lowered it!

Plus these smaller changes:

  • The game now warns you when pressing Escape or Start before quitting the match
  • More hints about where you can switch costumes in the game
  • The Unlockables button is now called Progress and has a percentage
  • Fixed rare crashes when pressing Start during a game
  • Fixed dangers from being in the screen the whole time (yes, that was a bug)
  • The other mall layouts now have their own floor tiles so they are more recognizable

The Clarity Update is live right now with build number 1046274.

We hope you enjoy it -- let us know what you think in the comments!

Fun, Fast-Paced Arcade Platformer

Super Flippin' Phones is a game where you run around a shopping mall knocking people's phones out of their hands, avoiding cops as you do so. The game includes multiple floors, having a unique power-up and class of mall cop, which gives the gameplay some variety. The pixel graphics look great, and suit the game; the music matches the same theme. The controls are simple and easy to learn, and the game also has controller support, which I thought was much easier to use than using a keyboard. The game doesn't really take itself seriously either; the developers gave the game a great sense of humour by adding adding things like cops wearing fairy costumes, and a heart-boxers costume.

PROS
  • Entertaining, challenging gameplay where standing still means death
  • Great sense of humour
  • Easy to learn
  • Lots of costumes to collect
  • Fun power-ups
  • Lots of enemy variety
  • High replayability

CONS
  • Not many different floors
  • The game is loud, and there is no option for volume adjustment
  • New costumes are too hard to pick up, as you have to stand still

Summary:
I really enjoyed Super Flippin' Phones, and will definitely play it again in future. The game's quirkiness keeps you coming back for more, and the gameplay keeps you hooked in. I'd definitely give this game a play, especially with such a cheap price tag. I also think it's worth mentioning that Super Flippin' Phones is available for Android.

Rating:
I'd give Super Flippin' Phones a 9/10.
