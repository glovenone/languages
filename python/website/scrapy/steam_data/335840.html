Goats on a Bridge is a cartoon styled runner game based on the Billy Goat Gruff fairy story. If you like the runner genre, where you have to get from A to B avoiding various obstacles, and picking up various objects that can be crafted into wearable equipment such as helments to customise your two characters, you'll probably like this. The game is still in early access, so hopefully we will see additional content added over the coming months, altohugh I reckon the dev's have already given us quite alot to be going on with. Apart from the main story mode there's a dash mode which has to be unlocked by playing the story mode; the dash mode is procedurally generated which means no two attempts will be the same.

This is definitely a game where you need to use a controller because you have to control both goats (Chubb and Tempo) at the same time. It is possible with the keyboard if you are very adept, but I found it almost impossible with the current key bindings for both goats. I do hope the dev's include the ability to change the key bindings in future updates before this game is fully released as I prefer using the keyboard to contoller. That said, i still enjoyed playing this game and will be keeping an eye on its further development. For an earel;y access game, this is very playable.

The Baaad
