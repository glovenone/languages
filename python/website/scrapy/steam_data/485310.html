Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“While game is not completly finished, it is ready for community to play it and help as shape the game based on their feedback. We don't want to spend our resources and money developing something community doesn't want to play. So it's better to hear player's voices before we make huge major updates we planned. We want to make a fun game people will enjoy, and the best way to do that is to let players help us achieve that.

Please remember that the game is currently in alpha stage and you may experience some issues or some of the features may not be implemented yet.

We recommend supporting us only if you want to help us develop better game!”

Approximately how long will this game be in Early Access?

“We are working on releasing it in approximately 5 months. Depends on the success of the game in Early Access, we might consider hiring more people to work on it, and finish it sooner.”

How is the full version planned to differ from the Early Access version?

“We are planning on releasing more game modes and maps. We will also continue working on all aspects of the game, which means we might change the current assets and gameplay features. We will try to shape the game based on our community feedback, so we still can't say exactly how the Full Released version of the game will look like.”

What is the current state of the Early Access version?

“Current state of the game is 'early beta'. Game is still in lack of content that will be available in full release but it is ready for open community tests.”

Will the game be priced differently during and after Early Access?

“Depends of the value we add into the game, the price may be slightly higher after game leaves Early Access.”

How are you planning on involving the Community in your development process?

“We don't want to make another multiplayer game that will die slowly after release and be unplayable because of lack of players. That's why we need to make a game people will actually play. We need the help of the community to make the game they want to play.”

About This Game

Take a role in a caroonish styled FPS multiplayer game. Choose among 5 characters and bunch of weapons and play Head Shot online with friends and strangers.

Features:
-5 classes (Assault, Sniper, Engineer, Medic, Machine Gunner);
-Various types of weapons including Assaults, Equipments, Machine Guns, Pistols, Rocket Launchers, Shotguns and Snipers;
-Vehicles;
-Different game modes (currently Death Match and Team Death Match)

Planned features:
-More weapons;
-More game modes;
-More items and various other content that will bring more fun into the game.
-Open chest system and various other stuff...


More features coming soon.
