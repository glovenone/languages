Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“We are currently doing a scheduled online multiplayer testing phase. If you are expecting a smooth online experience, you should wait until the game leaves Early Access.

We love retro racing games and we want to make sure that the release version will honor that heritage. We have a very experienced team who knows how to deliver a final and polished game. However, Race Online is a big and ambitious game including three major modules that are the single player, multiplayer and a track editor. Therefore, the contribution of the community is much appreciated especially because of the heavy support for community created content the game offers.

The single player content is mostly finished. New premade tracks will be added and old ones might be edited before the full release. However, that does not change the single player experience. In single player, a progression in the game will be balanced during the Early Access phase. Also AI players and vehicle upgrading will be balanced to make the single player even more fun to play.

Multiplayer is tested during the Early Access extensively. There simply is no other way to make a thorough testing how each vehicle work against each other in all tracks. If there exists some bugs in the networking code those are also found and crushed effectively.

The track editor is the same that was used to create all the current content of the game. This does not mean that the editor is perfect or free of bugs. The pipeline of making tracks can be improved, but it can be seen only when the community starts making their own awesome content to the game. That gives us a great feedback and the editor can be improved based on the experiences of players themselves. This feedback will not only include the editor itself, but also new objects and obstacles that could be added to the game making the game even more diverse as it is now.

Moreover, since the game is a competitive racing game, balancing all vehicles and tracks is vital for the enjoyable gameplay. There are a huge number of premade tracks in the game so there can still be something we have overlooked. A feedback from community, who have not stared the same levels thousands of hours will definitely make the game even better as a whole.”

Approximately how long will this game be in Early Access?

“A full release is planned for Fall 2016. However, if the game does not present the quality we want it to be the timeline can be expanded further.”

How is the full version planned to differ from the Early Access version?

“The Early Access version already contains all the key features of the game. The existing content will be polished and balanced to make the full release an outstanding bug free game. Currently there are nearly a hundred premade tracks in the Early Access version of the game, which will be expanded to hundred before the full release. The game will also receive some improvements to visual and sound effects during the Early Access period. Lastly, the online gameplay might include bugs, which will be found and fixed before the full release.

The game is not 100% polished, but all bugs that come along will be addressed as they come up.

The list to include all features in the full release is the following.
  • Hundred diverse tracks
  • Easy to use track editor
  • Steam Workshop support
  • Online & Local multiplayer up to 6 players
  • Extensive single player
  • Upgradeable vehicles
  • Multiple game modes
  • Stunning retro graphics brought to this millenium
”

What is the current state of the Early Access version?

“Most of the listed features have been implemented into the game and are already in the Early Access version. The online multiplayer part is missing for now, but is under heavy development and will be available in the near future. There are still polishing left to do and bugs to be fixed, but the game is fully playable and fun to play.
The Early Access version will be updated on a weekly basis and iterated upon that based on the feedback we get.

Also some of the Steam achievements are still not working or might be missing during the Early Access phase.”

Will the game be priced differently during and after Early Access?

“Nope, the price is the same as it will be. The game already includes all it's key features and main content that will be improved and increased before the full release. Also the game will receive free periodic content updates.”

How are you planning on involving the Community in your development process?

“We have been playing, modding and making games since the early 90's (feeling old!), so we know how important it is to be able to affect the game content. Therefore, we want not only to get feedback, but to give as good tools as possible for the community to make new content into the game. The track editor is specifically tailored to serve the community and provide tools to create new content with the same tools used by our development team. We want this to be easy and fun by integrating the tools to Steam Workshop and making a great editor for the game.

Needless to say, we hope to get feedback from everyone playing the game and hear all the things that needs improvement or fixing in the editor or in the game itself.”

About This Game

Race Online is the ultimate top down racing game with a strong retro feeling and super smooth gameplay. It is a fresh and evolved racing experience that seamlessly combines all the best parts from the good old racing games to the new.

Single player

Single player includes three different game modes. Single races are quick and fun to play at any time, but the real bread and butter of the single player mode are the lengthy careers that are unique for each vehicle in the game. On the other hand, the time trial mode is great for fine tuning driving lines towards the perfection, but also helps to achieve new lap records and to learn tricks and traits of each car in the game.

Multiplayer

Multiplayer can be played both online and locally up to six players. In the online multiplayer compete against players around the world in the fast paced tracks. Race Online is also a perfect game to play locally from one screen. Sit on a couch with your friends with gamepads, or gather around one keyboard to experience the retro racing as it was played back in the 90âs.

Editor

Track editor is easy to use and it includes all the same tools used by the developers. Creating custom tracks and sharing them with the community via Steam Workshop is fully integrated into the game. Initially there are 100 tracks in the game, but with the editor and community created content there are literally limitless amount of tracks to play.

Vehicles

The car selection includes different upgradeable vehicles from a serious sports car to a genuine UFO. Various color options expand the vehicle selection to 60 diverse options where to choose from. Extremely balanced gameplay keeps all vehicles competitive against each other. Learning the benefits of each vehicle is part of the fun that keeps the race interesting.

Gameplay

A polished gameplay is the key to every awesome game. A great amount of effort has been put into game controls. This way Race Online reaches the same quality of gameplay as it is in the best archetypes of this genre. Controls in Race Online are a result of careful consideration, thousands of testing hours and decades worth of experience from video games.

Early Access Review
Posted: 26 December, 2015
Race Online is a nice looking retro racer from top down perspective and is somewhat fun to play but if you're looking similar gameplay than in legendary Slicks 'n' Slide or Turbosliders you will be disappointed. The car handling is quite different and isn't what you would expect for a car as they move and handle like solid blocks, get stuck easily and you can't reverse you way out. There's not much difference between how cars (sports car, formula, tractor, etc.) handle and move. Also seems that track obstacle's or car's collision model isn't quite what the graphics show.

Overall the game feels ready and I didn't experience any bugs although the game crashed couple of times. The tracks are a mixed bunch, some are nice to race but often tracks are a bit annoying (due AI & collision model) and track profile isn't "professional" style. Some tracks look unfinished. The career mode is quite easy as the only difficulty is to get pass the AI cars after start and not to get stuck to them or pushed. Even on harder difficulty levels the difficulty comes from getting stuck to obstacles as the track is narrow and you can't pass or you crash to AI cars which are stuck. Also the grinding part of getting new cars and upgrading their parts feels artificial. The game ran quite nicely on my Macbook Pro (late 2013).

As the game is still in early access I hope the developers will enhance car physics and model before the release. They could take some example from e.g. Turbosliders which is fun to play but still competitive. Or maybe developers have played Turbosliders as the graphics and some of the tracks look similar.

Will be interesting to see how the multiplayer works when it's released.

In this development state I can't recommend this game. I hope the game gets better but I doubt it as the car physics are a big issue for games like these to change later on.

Early Access Review
Posted: 26 November, 2015
Update (4/27/2016): Online multiplayer was added to the game long after early adopters like me stopped playing it, so there's no-one to play with. The game just leaves you sitting in the lobby forever. It's such a shame, because it should be a lot of fun. Alas, I haven't had a single online race, yet. On a side note, it's a tad annoying to see the same developers release two other games (a pool one, and an ice-fishing one) while Race Online idles in Early Access. Perhaps when they officially release the game, mulitiplayer will see some action.

---

Single-player is great. It's fun, addictive and feels complete. I think all the bugs have been ironed out now.

You start with a choice of cars, race them in career mode over increasingly challenging tracks and earn in-game currency to upgrade your vehicles. Your AI opponents also upgrade their cars (or bikes, or tractors or UFOs!) so the game remains challenging even when you're fully upgraded. I feel like the AI is sometimes a bit too aggressive, but you can always restart a race if you get pushed off the track.

- works great with multiple xbox360 controllers
- no support for racing wheels, but the devs are considering it

In its current state, I recommend Race Online for single or *local* multiplayer only.
