Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“AWAKE is in Early Access for multiple reasons, the game is developed since 2014 and it is still in heavy development. The Early Access allows the community to play an important role in AWAKE development with suggestions and bug reports, thank's to the community we will be able to remove all the bad things and to implant things that you, players, want to see added to the game.

Please only purchase the game during this stage if you want to help support development and provide feedback and suggestions.

Alpha access also includes the full game when it releases and all future updates.”

Approximately how long will this game be in Early Access?

“Probably 1 year, everything depends on future feedback from all our players. We are going to work as hard and as long as it takes to make this game, a great game.”

How is the full version planned to differ from the Early Access version?

“We plan to add more of everything onto the full version, AWAKE will be more complete in multiple way, here is a list of some Planned Features:

- More Items
- More Crafts
- Better Graphics & Interface
- Bigger Island
- Diseases
- More Animals
- More Interactions with Nature”

What is the current state of the Early Access version?

“At the moment we have developed and implemented more than half the game. For example, some graphics (in game), the interface and some elements of gameplay will be improved, but thank's to Early Access you can see what the gameplay is like and how the game works.”

Will the game be priced differently during and after Early Access?

“Early Access version cost less than the final version because the game still needs some work. The price will still be cheap at the end of the early access, mainly because as an independant, I want everyone to be able to play this game.”

How are you planning on involving the Community in your development process?

“This is exactly why AWAKE is in early access, we want and we NEED feedback about the game, the community will help us to reference all the bugs so we will be able to fix them in the next updates and thanks to the feedback we will be able to know what you, players, want to see added in the game.”

AWAKE UPDATE (V1.3)

Hello everyone, the new update is now live !

Changelog:

- Video effects added, which result in a game optimisation.
- AWAKE is now more fluid
- The sensibility system has been adjusted in the pause menu
- The torch icon has been changed
- No more fire underwater ! Fire will disable himself if touched by water (fireplace and hand torches)
- The water level and the speed underwater have been adjusted
- New construction material available ! The coal is now mineable in game, it will allow you to create furnaces and torches (which is a new craft)


Coal source.

- Torch recipe added, you can now place torches from the crafting menu.


A crafted torch. (Not disabled by water for the moment)

- The console command "secret ressources" now add 100 coal to your inventory in addition to 100 wood log, 100 stones and 100 pieces of fabric.
- Crafted Furnaces can now be destroyed
- Melting option disabled when in front of a furnace

That's all for today ! Thank you all for supporting AWAKE !
Diios.

About This Game

Welcome to AWAKE, Mother nature is your biggest ennemy !

Your primary goal ? SURVIVE ! In AWAKE you are a shipwreck-survivor which managed to reach an island. Explore the island and tame the wilderness ! Find food and something to drink to stay alive, collect ressources and craft shelters, fireplaces, and many other things !

Please, remember that this game is in EARLY ACCESS, PLEASE BUY THIS GAME ONLY IF YOU WANT TO PARTICIPATE IN THE GAME DEVELOPMENT AND ONLY IF YOU ARE READY TO ENCOUNTER SOME BUGS.

System Requirements

    Minimum:
    • OS: Windows 7
    • Processor: Intel Dual-Core 2.4 GHz
    • Memory: 4 GB RAM
    • Graphics: NVIDIA GeForce 8800GT
    • DirectX: Version 9.0
    • Storage: 2 GB available space
    • Sound Card: DirectX®-compatible
    • Additional Notes: The minimal configuration in not the more optimised one and it is possible that you encounter lags or freeze.
    Recommended:
    • OS: Windows 7
    • Processor: Quad Core Processor
    • Memory: 6 GB RAM
    • Graphics: NVIDIA GeForce GTX 660
    • DirectX: Version 9.0
    • Storage: 2 GB available space
    • Sound Card: DirectX®-compatible
    • Additional Notes: The recommanded configuration is optimised to play AWAKE, you should not encounter lags or freeze.
