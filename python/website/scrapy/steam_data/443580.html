17 April

Patch Released

A new patch for Antenna has been released, along with it a basic user manual and updates to the System Requirements.

changes include:

  • Shortened Intro
  • New Framework/Runtime
  • Bug Fix on the Doorcode ** see note below
  • Full Screen Toggle
  • Many minor adjustments to improve performance

** Note, The conditions to clear the doorcode puzzle have slightly changed, set your positions and if correct when the last tone plays the puzzle will clear.

The Original Version of Antenna is still available by opting into it as a beta. This can be accessed by Right Clicking on Antenna in your library and Selecting Properties, then in the "Betas" tab you can select "beta" from the drop list to download the original version of Antenna.

About This Game

A machine ponders its loneliness. It scans the radio spectrum for an answer to its question.

Antenna is a short experience, an atmospheric expression.

Antenna currently supports US keyboard input only.
Other keyboard layouts will work but results may be unpredictable.
Mouse wheel required.
Puzzles are Audio Based, Please ensure you have sound on.

System Requirements

    Minimum:
    • OS: Windows 7
    • Processor: 2 GHz dual core
    • Memory: 4 GB RAM
    • Graphics: Hardware Accelerated Graphics
    • DirectX: Version 9.0c
    • Storage: 300 MB available space
    • Sound Card: Any but ideally Creative Sound Blaster or equivalent
    • Additional Notes: Requires Sound on to play
    Recommended:
    • OS: Windows 10
    • Processor: 2 GHz quad core +
    • Memory: 8 GB RAM
    • Graphics: Hardware Accelerated Graphics
    • DirectX: Version 10
    • Storage: 300 MB available space
    • Sound Card: Any but ideally Creative Sound Blaster or equivalent +
    • Additional Notes: Requires Sound on to play

This game is an easy free 100% game in less than an hour with a cool potential and some new ideas but is made porly on some different points

The Game
I personally don't like the game because of the idiocy from the puzzles. They are not very well made and very simple without any challenge, except that you have to be precise and its just a stupid way to be precise. But apart from the puzzles the game has, it does hold some potential. This game is of course very similar to limbo and probably has inspiration from it and I really liked limbo i think it was a really cool little indie game "which is free now". And with that said it can become a similar experience but with another "theme" if you can call it that. But if it has to be anything like limbo it really needs some work and another way of making puzzles or at least some major tweaking to the way they are. The story in itself also has some cool possibilities like it's the last robot on earth, or it has the inform the world about something and the list keeps going. But without any hints at all it really makes it uninteresting and boring. Not to say it has to be spelled in cardboard, but a story would be really cool and nice even if it was only minimal hints like things in the background or sounds, so you could imagine the other details for yourself.

Controls
The controls are a bit interesting and odd from other games since the way you activate antennas is with a somewhat random key on the keyboard. The idea is great but it is missing something a little crucial I think to make it work. It's a cool idea but since the keys that needs to be pressed are somewhat random it doesnt feel like the keys you press are of any importance. If they keys that needs to be pressed would be in a pattern or be similar to what you are doing in game it would work better but since its just some few random keys it loses some of it's effect. the movement is very simple and with the speed the game is it needs some more background things or something else to keep your attention because it's not very interesting and doesnt really holds your attention very well at least not for me.

Something REALLY IMPORTANT to change when it comes to controls is the "z/x" and "?" buttons to control with. As said before it's cool with some new buttons to press but it's a really bad choice with the "?" button. I found it frustrating to find the "?" button in game because when doing it in the windows search or on google or any other program it was "shift +" for me and it was not the same ingame. it was shift some random key on the keyboard. It didnt change to "shift +" when changing to english keyboard on the computer either. there is a reason all games pretty much have the same way of controlling because its those that are most effective and its was people are used to. but it's definitely possible to make some new controls with some other keys just not something like "?" that has the possibility to confuse and irritate.

Conclusion
so the game itself right now is just something you play for a bit and think "This needs some work", gets satisfied you finished it and then forgets about. with that said this game has some potential to be something interesting that you might remember like an cool experience. there is alot that needs to be reworked, adding a story and so on but the game has potential to be something cool but as it is now not the best.
