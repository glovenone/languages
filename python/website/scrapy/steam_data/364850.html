About This Game

This is a game where you control a ball that wishes to reach the sky. To help it achieve it's dream, you must complete a series of levels while collecting items. In a world of cubes and puzzles, you must be skillful or you will perish in the attempt.

FEATURES
Supports Xbox controller
8 bit Music
30 Levels

The Story
In a world where there were only boxes, one fell near a blue diamond and transformed into a ball. Now, in order to become a box again, it must collect all the white items to reach the blue diamond in the sky.

System Requirements

I wish this game was costing $10 more but was actually polished. There are gameplay issues, technical issues and just a lack of experience in my opinion. For starters, it doesn't support resolutions over 1080p. This game shipped July 2015. I'd expect higher resolution. The dyanamic shadow resolutions is absolutely horrendous and there is no settings to increase it. Performance wise it runs fine but changing v-sync will require a restart for it to apply properly. Worse part is there is no sensitivity setting, in a game that is all about 3D physics movement. Also you can't rebind your joystick axises. Finally some playtesting would have highlighted some pretty big flaws with the game where you end up in a broken state too often. To sum up, I was looking forward to a finished game and would have been happy to pay more but instead I got this.
