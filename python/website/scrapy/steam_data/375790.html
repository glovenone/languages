About This Game

Inspired by the classic DOS game Jump 'n Bump, Build 'n Bump retains all of the same head-stomping, multiplayer fun with a twist. You can build your own levels right before you play on them!

Featuring...

  • 2-6 players simultaneously (4 controllers, 2 on keyboard)
  • Local and online* multiplayer
  • The ability to save and load created levels
  • Five block types to build with (regular, spring, platform, spike, and dissolving)
  • Single-player practice mode against stationary targets
  • "Level master" mode where one player edits the level in real-time while everyone else fights

*Special Note

The online mode is limited to one connection between two computers. Familiarity with port forwarding is required; there is no dedicated server or lobby.

As much as I would agree with people that this game is stripped down to its bare concept, I must say that it is a very clever and innovative one. I am a huge fan of level editors, in fact I search the category regularly and it led me to this game. I fell that it is a shame how little content is provided, when with some more work, this could become a much greater game. The price was a responsible choice for the developers to make, considering the lack of a decent single player mode (which in their defence would be incredibly difficult to conceive in such a game). However, if you have a group of at least 3 friends, this is a fun and entertaining game to play, especially if youâre creative enough to play around with the limited objects it provides.

Once again, I would love to see updates to this game, because as it is, the concept has not been fully explored. Anyone could think of new and interesting things to add to this game, as a quick example, a block which gravity applies to, which can be pushed around and dropped off edges to land on players to kill them, but once again, this is down to the developers implementing these into the game.

I can fully understand that an online multiplayer mode should not be based on port forwarding, and should include a lobby, but people do need to understand that this is a very difficult goal to achieve and even requires money to be put into it, which may be whatâs holding this game back.

I have a lot of respect for this game and its concept, and can only suggest that everyone give it a fair chance, instead of spending minimal time in the single player mode and whining for your money back. However it should be noted that to experience the full game and all the fun you could have, you will require a local multiplayer couch party.
