While these two games are available on Steam, since we also have the original versions, this review is based on the original Super Nintendo and Steam games.

Bubsy Character

With so many mascot characters coming out back in the day, Bubsy tries to make himself stand out by using a wisecracking, smartass personality. Unlike the similarly-styled Gex that appeared during the mid 1990s, Bubsy restricts himself to using his comments only at the start of each level. This makes his attempts at humor more successful because they do not come out feeling as forced as Gexâs comments.

Bubsy I

As for the gameplay, it comes off as a striped-down version of Sonic the Hedgehog. Bubsy can run quickly making it that the player can cover a lot of ground in a short time. Additionally, when jumping, he can open his arms in order to glide across large gaps or dodge enemies. This comes in handy when dealing with dangerous situation. The problem with the game is that he dies after only one hit. There is an item that will protects you for a few seconds, but nothing that will stay until needed. Without anything to protect him, like Sonic with his speed, this will mean that you will get many cheap deaths. This only way to make it past a stage safely is to tap forward to prevent his speed from kicking. The one good thing about this is that the game is very generous with lives make it not so bad when dying. The game is still fun when moving slowly, but it becomes a hassle having to keep his speed down.

Bubsy II

Realizing that one-hit kills was a problem, they fix it by making that he needs three hits to die. Additionally, he can carry items that let me navigate the stages with some safety. On the surface, this sounds like a good thing. However, by giving him a safety measure, it makes the game easier. In fact, it is now too easy. With so many ways to protect the player, building up lives is no longer a problem. The game divides itself into two games, one labeled East Wing and one labeled West Wing. In both cases, I was able to finish the game with over 20 lives on my first try. Knowing that there are too many things on the player's favor make playing more of a snore than a joy.

Overall

The first game has enough basic elements for platforming fans to get into, but the second it not worth the effort. Just be grateful this collection stops before the disastrous 3D game. There is a reason there was never a fourth game. It is clear that the series is falling apart from the start. It would have been better if they included the other versions of the game instead like the Jaguar edition. If they did, the package might be more worth it.
