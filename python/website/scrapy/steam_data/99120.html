The game looked good. Was looking for a game like this. Lots of nice visuals. I like the concept of the game.

Would have been fun if the mouse controller did not over compensate my movements. If I tried to move quick, the target would slam from one side to the other. If I moved slow, I didn't get enough movement. I just did not feel I had control during the game. Not like in the menu. The game play controls with the mouse didn't feel right to me.

If the game works with a game pad, I will give that a try, I don't remember seeing settings for that though.

Some like the game, I don't recommend it, but that is do to my experience with it.
To me, even the old StarFox on the SNES is better. The controls is real nice on that game.

The intro looked good, though ran a little to fast for me to read what was being said. I would have to grab screen captures stills and read form those.

Over all, not horrible, but I couldn't play well.

extra:

Just tried this with a control pad. The GameStop Afterglow PS3/PC controler. And it worked! And the game controled like a dream. For one time.

On the first try, the mane gun fierd by pressing the right stick button. this I did not like. It was the only thing I did not like. Otherwise it played great! Better than expected.

Then I met with a porb. I tried to configuer the controler with in steam, and the controler was no longet reconised by the game, as a frist player controler. With the right triger, it stared player two. And I could move player two, but I had no wepons for player two.

So in short, this game has hidden functions for game pad control, not avalable in any settup within any options. This game seriously needs that!

I feel this game would rock, if it was fixed with options to not only configuer the keybord controls, but also updare the game with the option to use a game pad, and alow configueration of the game pad as well.

If this game is ever updated so the game pad can be fully used, I will not feel like I wasted my money on this game, and would fully suport giving this game a big thumbs up!
